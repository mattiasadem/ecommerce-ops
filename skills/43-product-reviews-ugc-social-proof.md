## Common pitfalls (18 from real builds)

1. **Pitfall #1 — Installing reviews-platform without configuring reviews-solicitation-timing.** The default reviews-platform installation sends the first review-request 30 days post-delivery, which means most customers have forgotten the order by then. Reviews-velocity caps at 0.3-0.5% of orders/mo instead of the canonical 1-3%. **Fix:** Configure reviews-solicitation-timing at day 7 + day 14 + day 30 post-delivery (Path A DEFAULT day 14 / Path B DEFAULT day 7 + day 14 + day 30 / Path C DEFAULT day 1 + day 7 + day 14 + day 30 + day 60). The Yotpo-2024 + Judge.me-2024 + Loox-2024 + Stamped-2024 + Okendo-2024 + Junip-2024 + REVIEWS.io-2024 + Trustpilot-2024 + Bazaarvoice-2024 + Pixlee-2024 + Olapic-2024 + Dash-Hudson-2024 + EmbedSocial-2024 + Flowbox-2024 + Tastewise-2024 + Nordot-2024 + CreatorIQ-2024 + Reeview-2024 + Feefo-2024 benchmarks show the canonical 3-touch cadence (day 7 + day 14 + day 30) drives 1.5-2.5× the reviews-velocity of the default day-30-only cadence.

2. **Pitfall #2 — Installing reviews-platform without removing reviews-submission-friction.** The default reviews-submission form requires 4+ fields (name + email + rating + title + body + photo + video) which drops submission-rate to 5-10% of opens. Reviews-velocity caps at 0.3-0.5% of orders/mo. **Fix:** Configure reviews-submission-friction-removal at 1-click-review-link + 1-click-photo-upload + 1-click-video-upload + 1-click-UGC-submission + 1-click-Google-Shopping-rating + 1-click-Meta-ads-rating (90%+ friction-removed DEFAULT). The Yotpo-2024 + Judge.me-2024 + Loox-2024 + Stamped-2024 + Okendo-2024 + Junip-2024 + REVIEWS.io-2024 + Trustpilot-2024 + Bazaarvoice-2024 + Pixlee-2024 + Olapic-2024 + Dash-Hudson-2024 + EmbedSocial-2024 + Flowbox-2024 + Tastewise-2024 + Nordot-2024 + CreatorIQ-2024 + Reeview-2024 + Feefo-2024 benchmarks show 1-click-review-link drives 3-5× the submission-rate of the default 4+-field form.

3. **Pitfall #3 — Installing reviews-platform without configuring reviews-respond-rate-monitor.** The default reviews-platform has no respond-rate-monitor, so 1-star and 2-star reviews go unresponded, dragging CSAT and product-page-trust. Reviews-velocity caps at 0.5-0.8% of orders/mo. **Fix:** Configure reviews-respond-rate-monitor at 90%+ respond-rate on 1-star 2-star 3-star 4-star 5-star (Path B+ DEFAULT 90% / Path C DEFAULT 95%).

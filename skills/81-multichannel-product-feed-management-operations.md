## When to use this skill

Use this skill the first time any of these is true:

- the merchant has a **single product data source** (Shopify + BigCommerce + Salesforce Commerce Cloud + Akeneo + Salsify + ERP) but wants to distribute it to **10+ downstream feed-targets** (Google Merchant Center + Meta Catalog + TikTok Catalog + Pinterest Catalog + Snap Catalog + Bing Product Ads + Walmart Feed + Amazon Selling Partner Feed + eBay Seller Center Feed + Affiliate Feed + Retail-Media Feed + Criteo + RTB House) and has no documented normalized feed-orchestration layer to translate schema, enrich attributes, map taxonomy, route inventory, arbitrate price, enforce policy, monitor feed-health, and recover from disapprovals;

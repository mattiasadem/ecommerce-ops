# TikTok Shop / live commerce

> Move #15.x is the canonical **shop-native-creator-led social-commerce channel layer** for the Shopify-DTC stack — the move that lets a $500k+ GMV brand capture the **5–20% of US social-commerce GMV that TikTok-Shop is on track to control by 2026** ($70B+ US TikTok-Shop-GMV 2026 vs $25B in 2024 = 2.8× YoY compounding per eMarketer 2024 + Insider Intelligence 2024 + Jungle Scout 2024 benchmarks) through a **5-pillar build**: (Pillar 1) **TikTok-Shop-Seller-Center + category-approval + product-feed-optimization** — the platform-foundation layer with 10 permitted-categories (Beauty / Fashion-Accessories / Home / Electronics / Food / Pet / Baby / Health / Outdoor / Toys-and-Collectibles), 1-4-week category-approval-timeline, 30-50% first-time-applicant-rejection-rate, and the canonical per-SKU product-feed-discipline (≤100-char title + 1:1 ≥800px image + 3-5 in-use images + keyword-rich description + $15-$45 hero-SKU price-tier + 5-15% shipping-subsidy); (Pillar 2) **creator-affiliate + 5-payout-structures + TikTok-Shop-Creator-Marketplace** — the canonical 4-channel creator-affiliate-pool build (TikTok-Shop-Creator-Marketplace + Aspire + Collabstr + Instagram-hashtag-scrape per Move #15) yielding 20-50 onboarded-creator-affiliates and the canonical 5-payout-structures (CPM $10-$30/1000-views / CPS 10-25%-of-GMV / flat-fee $200-$2k/creator/post / hybrid 5%-of-GMV + $200-base-fee / product-seeding-only); (Pillar 3) **shoppable-video-ads + Spark-Ads-boost + In-Feed-Ads-with-Product-Module + Top-View-Ads** — the paid-amplification layer achieving 2-4× ROAS for shoppable-video-ads vs in-feed-ads-without-product-module + 1.5-2× ROAS for Spark-Ads-only per TikTok-for-Business-2024 benchmarks; (Pillar 4) **LIVE-shopping + 5-segment-show-runner-script + 4-hour-week-cadence** — the live-commerce layer achieving 5-15% conversion-rate vs 1-3% for shoppable-video-ads (3-5× higher CVR per NRF 2024 + Modern Retail 2024 + Insider Intelligence 2024 benchmarks) with $500-$2k studio-build (Ring-light + iPhone-15-Pro + tripod + lavalier-mic + 1080p-webcam + backdrop + teleprompter-app + Streamlabs-OBS or TikTok-LIVE-Studio) and the canonical 5-segment-show-runner (intro / demo / Q&A / creator-guest-takeover / closing-flash-sale); ## When to use this skill

**Use this skill when the operator has shipped the canonical retention-substrate + has Gen-Z/Millennial audience skew or creator-led-brand-voice + is ready to add a shop-native-creator-led social-commerce channel.** TikTok-Shop / live-commerce is structurally distinct from Move #15 (creator-affiliate) + Move #15.x-publisher-affiliate + Move #16 (creator-economy expansion) + Move #17 (Pinterest-organic-discovery) — it is the only channel that operates INSIDE TikTok's native shop-tab + LIVE-stream + shoppable-video-ads stack — and the operator should only initiate it when ALL of the following are true:

- **DTC-retention-substrate steady-state ≥90 days.** Move #1 (cart-abandon) + Move #4 (welcome) + Move #6 (Triple Whale attribution) + Move #7 (SMS) + Move #8 (loyalty) + Move #11 (subscriptions) + Move #15 (creator-affiliate) are all live and producing measurable revenue. Without this 90-day steady-state baseline, the operator lacks the cohort-LTV reference frame needed to measure TikTok-Shop-driven-cohort vs organic-DTC-cohort vs paid-Meta-cohort — a brand without TikTok-aware attribution undercounts TikTok-driven-DTC-attribution by 30-50% per Triple Whale 2024 benchmarks. Verify: Triple Whale `cohort_ltv_dtc_baseline` returning ≥$120 LTV per Move #1 + #4 + #6 + #7 + #8 + #11 + #15 cohort.
- **≥$500k US DTC GMV for 3 consecutive months.** TikTok-Shop requires ≥10 SKUs with ≥25% margin-headroom (TikTok-Shop-takes 8% commission + creator-affiliate 10-25% of GMV + shipping-subsidy 5-15% of AOV + LIVE-show production $50-$200/session = 30-50% of GMV cost-stack). Brands below $500k GMV should defer until DTC retention is mature OR run Path A creator-affiliate-only. Verify: Triple Whale monthly-GMV-by-source dashboard returning ≥$42k US DTC GMV per month for 3 consecutive months.
- **≥10 SKUs with ≥25% TikTok-Shop-margin-headroom.** TikTok-Shop-permitted-categories require ≥10 SKUs for category-approval-baseline (brands with <10-SKUs need to bundle-Kits to reach threshold OR defer). Wholesale-discount-tier-matrix requires ≥25% margin-headroom for TikTok-Shop-creator-affiliate-commission + shipping-subsidy. Verify: `scripts/unit_economics_check.py --gross-margin --tiktok-shop-takes --creator-affiliate-commission` returns margin-headroom ≥25%.
- **≥40% Gen-Z / Millennial audience skew (18-34) OR creator-led-brand-voice.** TikTok-user-base is 70%+ age 18-34 per eMarketer 2024 + Jungle Scout 2024 benchmarks. Brands with 40%+ audience 18-34 see 3-5× higher TikTok-Shop-conversion vs brands with <20% audience 18-34. Brands without Gen-Z/Millennial skew but with strong creator-led-brand-voice (sustainability / luxury / niche-craft / beauty / wellness) also fit. Verify: Triple Whale audience-demographics or Shopify Analytics → Customers → Age distribution returning ≥40% in 18-34 bracket OR creator-engagement-rate on existing creator-affiliate-content ≥3%.
- **3PL OR in-house warehouse capable of 95%+ ship-on-time-rate.** TikTok-Shop-SLA requires ≥95% ship-on-time-rate per Shop-Score-2024-metric (Shop-Score is the algorithmic-distribution-bonus driver). Brands with sub-95% baseline need to fix DTC-fulfillment before TikTok-Shop-launch OR move to 3PL (Move #12). Verify: Shopify Shipping analytics or 3PL-portal showing ship-on-time-rate ≥95% over last 30 days.
- **8-prereq TikTok-Shop-onboarding-pack documented.** (1) TikTok-Business-Account (free, business-email + business-registration-document), (2) category-approval-application per TikTok-for-Business-2024-category-approval-list (Beauty / Fashion-Accessories / Home / Electronics / Food / Pet / Baby / Health / Outdoor / Toys-and-Collectibles — each with separate documentation [brand-registration + product-safety-cert (CPSIA for baby OR FDA-registration for beauty/food/supplements OR FCC-certification for electronics OR CPSC-registration for general-merchandise) + product-insurance $1M-$2M general-liability + $1M product-liability + category-specific-test-results + warehouse-safety-cert + brand-authorization-letter]; approval-timeline 1-4 weeks; rejection-rate 30-50% first-time-applicants), (3) Shop-Score-baseline-audit (current-DTC ship-on-time-rate + return-rate + chat-response-rate + positive-review-rate to project Shop-Score-baseline; brands with <4.0-baseline-Shop-Score need to fix DTC-operations first), (4) creator-affiliate-pool-baseline (≥20 ready-to-onboard via TikTok-Shop-Creator-Marketplace + Aspire + Collabstr + Instagram-hashtag-scrape per Move #15; brands with <5-pool-baseline should defer until pool is built), (5) LIVE-shopping-studio-build ($500-$2k one-time [Ring-light + iPhone-15-Pro + tripod + lavalier-mic + 1080p-webcam + backdrop + teleprompter-app + Streamlabs-OBS or TikTok-LIVE-Studio $0/mo] OR outsource-to-TikTok-LIVE-Production-Partner $200-$500/session), (6) TikTok-Shop-product-feed-optimization (per-SKU TikTok-Shop-product-title ≤100-chars + product-image 1:1-square-≥800px + 3-5-additional-images showing product-in-use + product-description-with-keywords [SEO + benefit-led + ingredient-led + use-case-led] + price-tier-fits-TikTok-Shop-impulse-pattern $15-$45 hero-SKU + 5%-15%-shipping-subsidy), (7) Klaviyo-TikTok-channel-integration ($0 with Klaviyo-Standard; $45/mo with Klaviyo-Email-and-SMS for TikTok-Shop-cart-abandon + TikTok-Shop-welcome-flow + TikTok-Shop-post-purchase + TikTok-Shop-subscriber-cohort-LTV), (8) Triple-Whale-TikTok-attribution-setup (Triple-Whale-app-install $179/mo Starter or $1,290/mo Pro + TikTok-cohort-overlay-wire + post-purchase-survey with TikTok-source-tracking).
- **4-8 hr/wk operator-capacity during Phase 1+2.** Phase 1 (Seller-Center onboarding + category-approval + 20-50 creator-affiliate onboarding) requires 4-6 hr/wk for 2 weeks. Phase 2 (shoppable-video-ads launch + creator-cohort-LTV measurement) requires 4-6 hr/wk for 4 weeks. Phase 3 (LIVE-shopping-launch + TikTok-Shop-affiliate-program-launch) requires 4-8 hr/wk for 6 weeks (LIVE-show-running is time-intensive). Phase 4 (steady-state + Shop-Score-audit + LIVE-cadence-optimization) requires 2-4 hr/wk ongoing. Verify: operator has committed 4-8 contiguous hr/wk for 8-16 weeks.

**Don't use this skill when:**

- **<$500k US DTC GMV OR <10 SKUs OR no Gen-Z/Millennial audience skew OR <25% margin-headroom.** Below these thresholds, TikTok-Shop's structural-upside is muted and the build-cost ($500-$2k studio + 30-50% of GMV takes) exceeds the incremental-TikTok-Shop-GMV ceiling. Defer until DTC retention-substrate is mature + SKU-broadness + audience-skew + margin-headroom all cross threshold.
- **No LIVE-shopping-studio-capacity (no operator-time for 4 hr/wk).** Phase 3 LIVE-shopping-launch is the highest-ROI layer (5-15% CVR vs 1-3% for shoppable-video-ads) but requires 4-8 hr/wk operator-time. Brands without this capacity should run Path A creator-affiliate-only + shoppable-video-ads (no LIVE) at 4.5:1 ROI vs Path B 8.5:1 ROI.
- **Brand-voice violates MAP-policy on TikTok-Shop-creator-affiliate-direct-discounts.** TikTok-Shop-creator-affiliate often discount-prominently with limited-time-offers (TikTok-Shop-flash-sale pattern) that violate MAP-policy per Faire 2024 + Ankorstore 2024 cross-reference. Luxury brands protecting MAP-policy should use TikTok-Shop-affiliate-program-with-MAP-policy-guardrails (separate from creator-affiliate-direct-discounts) or defer TikTok-Shop until MAP-policy-enforcement-tooling exists.
- **Pre-launch OR <30 days post-launch.** TikTok-Shop requires a stable DTC baseline (Move #1 + Move #4 + Move #6 + Move #7 + Move #8 all ≥30 days shipped). Without this baseline, the cohort-LTV-measurement is too noisy.
- **3PL ship-time-baseline >2 days OR ship-on-time-rate <95%.** TikTok-Shop-SLA requires 95%+ ship-on-time-rate per Shop-Score-2024-metric. Brands with sub-95% need to fix DTC-fulfillment first (3PL migration Move #12 or in-house warehouse optimization).
- **Restricted TikTok-Shop-categories (Firearms / Alcohol / Tobacco / Cannabis / Pharmaceuticals / Adult-Content / Gambling / Financial-Products).** These are NOT in the TikTok-Shop-permitted-categories list per TikTok-for-Business-2024-category-approval-list. Brands in these verticals must use other channels.

## The build (4 phases over 8–16 weeks)

### Phase 1 — TikTok-Shop-Seller-Center-onboarding + category-approval + first-50-creator-affiliate-onboarding (Weeks 1-2)

**Goal:** 4.5+ Shop-Score baseline + 20-50 active creator-affiliates + $25k-$100k first-quarter TikTok-Shop-GMV.

**Build steps:**

1. **Set up TikTok-Business-Account** (Day 1, 1-3 hours). Create free TikTok-Business-Account at business.tiktok.com with business-email + business-registration-document. Verify account in 1-3 days.

2. **Apply for TikTok-Shop-Seller-Center category-approval** (Days 1-7, 4-8 hours). Apply at seller-center.tiktok.com with category-specific-documentation for the brand's top SKU category:
   - **Beauty / Skincare / Haircare / Cosmetics** — brand-registration + product-safety-cert (FDA-registration for cosmetics) + product-insurance $1M-$2M + warehouse-safety-cert (FDA-registered) + brand-authorization-letter.
   - **Fashion-Accessories / Apparel** — brand-registration + product-safety-cert (CPSIA for baby-apparel OR CPSC-registration for general-merchandise) + product-insurance + warehouse-safety-cert + brand-authorization-letter.
   - **Home-Decor / Furniture / Kitchen** — brand-registration + product-safety-cert (CPSC-registration) + product-insurance + warehouse-safety-cert + brand-authorization-letter.
   - **Electronics** — brand-registration + product-safety-cert (FCC-certification) + product-insurance + warehouse-safety-cert + brand-authorization-letter.
   - **Food / Beverages / Supplements** — brand-registration + product-safety-cert (FDA-registration) + product-insurance + warehouse-safety-cert (FDA-registered) + brand-authorization-letter + product-safety-data-sheet + microbial-test-results + stability-test-results.
   - **Pet** — brand-registration + product-safety-cert + product-insurance + warehouse-safety-cert + brand-authorization-letter.
   - **Baby** — brand-registration + product-safety-cert (CPSIA + CPSC-registration) + product-insurance + warehouse-safety-cert + brand-authorization-letter.
   - **Health** — brand-registration + product-safety-cert (FDA-registration for supplements) + product-insurance + warehouse-safety-cert (FDA-registered) + brand-authorization-letter.
   - **Outdoor** — brand-registration + product-safety-cert + product-insurance + warehouse-safety-cert + brand-authorization-letter.
   - **Toys-and-Collectibles** — brand-registration + product-safety-cert (CPSIA + CPSC-registration) + product-insurance + warehouse-safety-cert + brand-authorization-letter.
   - Approval-timeline: 1-4 weeks. Rejection-rate: 30-50% first-time-applicants.

3. **Optimize TikTok-Shop-product-feed** (Days 7-14, 8-12 hours per 10 SKUs). Per-SKU TikTok-Shop-product-listing:
   - TikTok-Shop-product-title ≤100-chars (e.g. "Vitamin C Serum for Brightening — 30ml Glass Dropper Bottle — Vegan + Cruelty-Free").
   - Product-image 1:1-square ≥800px (hero-image on white-background).
   - 3-5 additional-images showing product-in-use (lifestyle-shots + benefit-shots + ingredient-shots + creator-content-shots).
   - Product-description-with-keywords (SEO-led + benefit-led + ingredient-led + use-case-led; 200-500 chars).
   - Price-tier-fits-TikTok-Shop-impulse-pattern ($15-$45 hero-SKU + $50-$200 wholesale-eligible secondary-SKU).
   - 5%-15%-shipping-subsidy (TikTok-Shop-product-listing-default-offers-shipping-subsidy; brand can adjust).
   - TikTok-Shop-product-card-preview (test on TikTok-mobile-app before publish).

4. **Build creator-affiliate-pool-baseline** (Days 7-14, 8-12 hours). Onboard 20-50 creator-affiliates via 4-channel pool-build:
   - **TikTok-Shop-Creator-Marketplace** (free, 10-20 creators): Apply at creator-marketplace.tiktok.com, browse-creator-profiles by category + audience-size + engagement-rate, send-outreach with creator-brief-template (see TikTok-Shop-Creator-Brief-Templates-2024).
   - **Aspire** ($250-$2k/mo, 5-10 creators): Apply at aspire.io, browse-creator-profiles by category + audience-size + engagement-rate, request-influencer-partnership.
   - **Collabstr** ($0-$500/mo, 5-10 creators): Apply at collabstr.com, browse-micro-creator-profiles ($50-$500/post), request-creator-collab.
   - **Instagram-hashtag-scrape** (free, 5-10 creators): Scrape-instagram-hashtags for category-relevant creators (#ecobeauty + #skincareroutine + #cleanbeauty etc.); manually-outreach via DM with creator-brief-template.

5. **Set up creator-affiliate-payout-structures** (Days 7-10, 2-4 hours). Document the canonical 5-payout-structures per creator-tier:
   - **CPM** ($10-$30/1000-views) — for nano-creators with high-view-count + low-CVR.
   - **CPS** (10-25%-of-GMV) — for micro + mid-tier creators with proven-conversion-rate.
   - **Flat-fee** ($200-$2k/creator/post) — for macro-creators with brand-awareness-focus.
   - **Hybrid** (5%-of-GMV + $200-base-fee) — for mid-tier creators with both brand-awareness + conversion-focus.
   - **Product-seeding-only** ($0 + free-product) — for early-stage creator-relationships + UGC-content-generation.

6. **Set up Klaviyo-TikTok-channel-integration** (Day 10, 1-2 hours). In Klaviyo, integrate TikTok-channel at Settings → Integrations → TikTok → Connect. Verify TikTok-Shop-cart-abandon event is firing on test-order.

7. **Set up Triple-Whale-TikTok-attribution** (Day 10, 2-4 hours). In Triple-Whale, install TikTok-Shop-pixel on TikTok-Shop-Seller-Center, set up TikTok-cohort-overlay, configure post-purchase-survey with TikTok-source-tracking-question ("How did you hear about us?" with TikTok + TikTok-Shop + LIVE-Shopping + Spark-Ads + creator-content options).

**Phase 1 exit gate:** 4.5+ Shop-Score baseline + 20-50 active creator-affiliates + Klaviyo-TikTok-channel-integrated + Triple-Whale-TikTok-attribution-wired + first-10-SKUs-listed-and-optimized.

## Common pitfalls (15 from real TikTok-Shop / live-commerce builds)

1. **No category-approval-documentation-pre-staged.** Submitting category-approval-application with missing-docs (brand-registration-cert + product-safety-cert + product-insurance + warehouse-safety-cert + brand-authorization-letter) → 4-week approval-timeline OR 30-50% rejection. Fix: pre-stage all required docs before applying; verify 1-2-week approval.
2. **Unoptimized TikTok-Shop-product-feed.** Generic Shopify-product-feed (no per-SKU TikTok-optimization) → 1-2× CVR vs 2-4× CVR for optimized-feed. Fix: per-SKU TikTok-Shop-product-title ≤100-chars + 1:1 ≥800px image + 3-5 in-use images + keyword-rich description + $15-$45 hero-SKU price + 5-15% shipping-subsidy.
3. **Sub-95% ship-on-time-rate baseline.** TikTok-Shop-SLA requires ≥95% ship-on-time-rate per Shop-Score-2024-metric. Brands with sub-95% baseline lose Shop-Score-algorithmic-distribution-bonus. Fix: ship-on-time-rate-audit before TikTok-Shop-launch; 3PL-migration (Move #12) if in-house-warehouse can't hit 95%.
4. **Creator-affiliate-pool-baseline <5.** Launching TikTok-Shop with empty-creator-affiliate-pipeline (<5 creators) → no organic-TikTok-content-driving-Shop-traffic. Fix: build-creator-affiliate-pool-baseline ≥20 BEFORE TikTok-Shop-launch (TikTok-Shop-Creator-Marketplace + Aspire + Collabstr + Instagram-hashtag-scrape).
5. **Single payout-structure (CPS-only or flat-fee-only).** All-creators-on-same-payout-structure → miss per-voice + per-tier optimization. Fix: 5-payout-structures per-voice matrix (Default 60% product-seeding + 25% micro-CPS 15-20% + 10% mid-tier-CPS 20-25% + 5% macro-flat-fee).
6. **No Spark-Ads-strategy.** Boosting all-creator-content (spend on low-performers) OR no Spark-Ads at all (misses organic-content-paid-amplification). Fix: boost-only-top-10%-performing-organic-creator-content (CTR >2% OR CVR >1.5% OR 10k+ views); boost-budget $50-$500/day per audience-segment; refresh-creative-weekly.
7. **No Triple-Whale-TikTok-cohort-overlay.** No-cohort-LTV-measurement → vanity-metrics-only (impressions + clicks + GMV) without per-creator-cohort-LTV + per-channel-cohort-LTV. Fix: Triple-Whale-TikTok-cohort-overlay-wire before shoppable-video-ads-launch; weekly-review-of-TikTok-Shop-driven-cohort-LTV vs organic-DTC-cohort-LTV vs paid-Meta-cohort-LTV.
8. **LIVE-shopping-studio quality below threshold.** iPhone-handheld + bad-lighting + bad-audio → LIVE-stream-conversion-rate <2% (vs 5-15% benchmark). Fix: invest $500-$2k in studio-build (Ring-light + iPhone-15-Pro + tripod + lavalier-mic + 1080p-webcam + backdrop).
9. **No LIVE-cadence-discipline.** Daily-LIVE without A/B-test (operator-burnout) OR monthly-LIVE (loses algorithmic-distribution-bonus). Fix: 1 LIVE-session-per-week 60-90-minutes; A/B-test 1 vs 2 vs 3 sessions/week in Phase 4; optimize-cadence per data.
10. **Unscripted-LIVE without 5-segment-show-runner-script.** Generic-script (no per-segment-timing) OR ad-lib-LIVE-show → lower CVR. Fix: 5-segment-script with per-segment-timing (5min/15min/15min/15min/10min = 60min) + per-segment-talking-points + per-segment-CTA + per-segment-flash-sale-discount.
11. **No FTC-16-CFR-Part-255-disclosure-compliance.** Creator-content + paid-amplification-ads without #ad + #TikTokShopPartner hashtag → FTC-violation ($10k+ per-violation-fine). Fix: all-creator-brief-templates include disclosure-language; verify-disclosure-weekly.
12. **MAP-policy-violation on TikTok-Shop-creator-affiliate-direct-discounts.** TikTok-Shop-creator-affiliate often discount-prominently with limited-time-offers → MAP-policy-violation → 30-50% DTC-cannibalization per Faire 2024 + Ankorstore 2024 cross-reference. Fix: TikTok-Shop-affiliate-program-with-MAP-policy-guardrails (separate from creator-affiliate-direct-discounts); no-discount-promotion-clause OR approved-discount-amount-clause.
13. **Klaviyo-TikTok-channel-integration-misconfigured.** Single-TikTok-channel-flow (cart-abandon-only); no post-purchase-flow. Fix: Klaviyo-TikTok-channel-full-integration (cart-abandon + welcome + post-purchase + subscriber-cohort-LTV).
14. **TikTok-Shop-affiliate-program-launch-without-30-day-cookie-window.** Affiliate-program-without-cookie-window → affiliate-commission-conflicts (last-click vs first-click). Fix: 30-day-cookie-window per Move-#15-affiliate-program-benchmarks.
15. **No Triple-Whale-cohort-LTV-iteration-cycle.** Launching-and-forgetting (no weekly-review of creator-affiliate-cohort-LTV + no iteration-of-payout-structure). Fix: Triple-Whale-cohort-LTV-iteration-cycle-weekly (top-10-creator-cohort-LTV-leaderboard + iterate-payout-structure per the 5-payout-structures).

## The build (4 phases over 12–24 weeks, 4–8 hr/wk operator time)

### Phase 1 — Creator-discovery-platform-onboarding + first-50-micro-creator-UGC-outreach + Triple-Whale creator-cohort-LTV-Wire (Weeks 1–4, ~12 hr)

1. **Pick creator-discovery-platform per Path** (Path A: Collabstr free-tier OR Aspire SaaS-free-tier / Path B DEFAULT: Aspire SaaS $500-$2k/mo OR Aspire agency $500-$2k/mo / Path C: GRIN $2.5k+/mo OR CreatorIQ $2.5k+/mo). For $500k-$5M GMV brands the canonical pick is Aspire SaaS (Path B DEFAULT) — Aspire is the SMB-mid-market-creator-discovery-platform per Aspire 2024 + CreatorIQ 2024 benchmarks.
2. **Wire Triple-Whale creator-cohort-overlay** (Triple Whale $179/mo Starter OR $1,290/mo Pro + creator-cohort-overlay-wire + post-purchase-survey with creator-source-tracking question "How did you first hear about us?" with options [TikTok creator / Instagram creator / YouTube creator / podcast / Substack / UGC / other]). **Verify** post-purchase-survey appears on test-order BEFORE Phase 2 (Pitfall #11).
3. **Draft content-licensing-template with legal counsel** (canonical 3-clause agreement: perpetual-organic-usage-rights + 90-day-paid-amplification-rights + creator-attribution-required). Templates in `assets/20-creator-discovery-templates.md` per the playbook-tick follow-up. **Verify** with legal counsel that the template covers US + EU + UK jurisdictions if international.
4. **Draft whitelisting-ads-template with legal counsel** (canonical whitelisting-ads-agreement: creator-grants-brand-permission-to-run-creator-content-as-brand-paid-ads via Meta Brand Collabs Manager + TikTok Creator Marketplace + YouTube Shorts Paid Discovery + Pinterest Creator Reveal).
5. **Draft FTC-compliance-disclosure-language-SOP** (16 CFR Part 255 — `#ad` / `#sponsored` / `#partner` / `#gifted` must appear on every creator-content-post per FTC 2023-2024 enforcement; $10k+/violation fine).
6. **Onboard first 50 micro-creators** (1k-10k-followers per Move-#15-affiliate-program-benchmarks) via Collabstr or Aspire product-seeding-only payout-structure ($0 + free-product + shipping-subsidy $5-$50/micro-creator). Filter by audience-demographics-fit (≥40% audience 18-34) + content-quality-baseline (last-10-posts avg-views >1k) + brand-safety (no adult / political / controversial content).
7. **Build Klaviyo creator-content-flows** (creator-content-welcome-flow [first-purchase-after-creator-attribution → "Welcome from [Brand] — your creator [X] sent you" personalized flow] + creator-driven-cart-abandon [cart-abandon-from-creator-attribution → different copy referencing creator-source] + creator-driven-post-purchase [post-purchase with creator-storytelling + creator-cross-sell]).

### Phase 2 — Mid-tier-creator-onboarding + content-licensing-launch + Triple-Whale creator-content-cohort-LTV-measurement (Weeks 5–10, ~18 hr)

1. **Onboard first 20 mid-tier-creators** (10k-100k-followers) via Aspire with mid-tier-CPS 20-30%-of-GMV + $100-base-fee + content-licensing-rights.
2. **Execute content-licensing-template with all 50+ creators** (every creator must sign the content-licensing-agreement before posting brand-related content). **Verify** via Aspire / Collabstr / GRIN signed-document-archive.
3. **Execute whitelisting-ads-template with all 50+ creators** (every creator must sign the whitelisting-agreement before brand-pays-for-paid-amplification). **Verify** signed-document-archive.
4. **Wire Triple-Whale creator-content-cohort-LTV** (creator-content-cohort-overlay + weekly-creator-content-cohort-LTV-report at 30/60/90-day windows: creator-content-driven-cohort vs organic-DTC-cohort vs paid-Meta-cohort).
5. **Launch first 100 creator-content-posts** with FTC-compliance-disclosure-language-verified on 100% of posts (per creator-economy-FTC-compliance-SOP).
6. **Verify first $50k-$250k creator-economy-driven-revenue** via Triple-Whale-creator-cohort-overlay + creator-discovery-platform-analytics.
7. **Cross-pollinate with Move #15 affiliate-program** (creator-economy-creators-get-affiliate-program-codes; affiliate-program-publishers-get-creator-content-licensing). For Path B, this is the structural mechanism that compounds Move #15 ROI.

### Phase 3 — Macro-creator-flagship-campaign + creator-affiliate-program-bridge + GRIN-or-CreatorIQ-enterprise-CRM (Weeks 11–18, ~25 hr)

1. **Onboard first 5-15 macro-creators** (100k+-followers) via Aspire + GRIN/CreatorIQ with macro-flat-fee $500-$5k/creator/campaign + content-licensing + whitelisting-rights + cross-platform-post (TikTok + Instagram + YouTube + Substack).
2. **Build creator-affiliate-program-bridge** (each macro-creator gets unique-affiliate-program-code + Triple-Whale-cohort-overlay so creator-driven-affiliate-attribution is measurable per Move #15 Pillar 4 cohort-LTV-measurement).

For Path B $500k-$5M brands this is overkill — defer to Path C thresholds.
4. **Launch first 4 macro-creator-flagship-campaigns** (each with documented creator-economy-program-ROI ≥5:1 per Pillar 5 measurement discipline).
5. **Build creator-tier-promotion-SOP** (4-trigger SOP [volume + cohort-LTV + content-quality + tenure] for promoting micro-to-mid-tier + mid-tier-to-macro per Move-#15-affiliate-program-tier-promotion-pattern).

7. **Cross-pollinate with Move #15.x TikTok-Shop** (creator-economy-creators-get-TikTok-Shop-affiliate-codes; TikTok-Shop-creators-get-creator-content-licensing). For Path B brands with TikTok-Shop-launch-ready, this compounds Move #15.x ROI.

## Common pitfalls (15 from real creator-economy builds)

**A. Creator-discovery-platform-selection pitfalls**

1. **Selecting the wrong creator-discovery-platform for the brand's GMV-tier.** The canonical anti-pattern: $200k-GMV brand launches full-GRIN-enterprise-CRM ($2.5k+/mo) before reaching Path-B-thresholds, caps at $50k-first-quarter-creator-driven-revenue, loses $30k+ on over-investment. **Fix:** Match creator-discovery-platform to GMV-tier per the canonical 5-path-decision-matrix in Pillar 1; default to Path B for $500k-$5M brands (Aspire SaaS or agency); Path A for $100k-$500k brands (Collabstr or Aspire-free-tier); Path C only for $5M+ brands with dedicated-creator-economy-team (GRIN or CreatorIQ).

2. **Building creator-portfolio on macro-creators-only ("the 1-mega-creator-bet anti-pattern").** The canonical anti-pattern: brand signs 1 mega-creator with 1M-followers for $5k-flat-fee + content-licensing + whitelisting, achieves $10k-creator-driven-revenue, then the creator-pivots-to-other-brand. **Fix:** Build canonical 60% micro + 30% mid-tier + 10% macro creator-tier-mix per Move-#15-affiliate-program-benchmarks; macro-creators should be 5-15 not 1-3; brands with 100+ active creators should consider GRIN or CreatorIQ for enterprise-CRM.

**B. Creator-payout-structure pitfalls**

3. **Selecting creator-payout-structure that doesn't match creator-tier.** The canonical anti-pattern: brand offers macro-flat-fee $5k/campaign to micro-creators (1k-10k-followers) who expect product-seeding-only $0 + free-product; brand offers product-seeding-only to macro-creators (100k+-followers) who expect macro-flat-fee + content-licensing + whitelisting. **Fix:** Match payout-structure to creator-tier per the canonical 5-payout-creator-economy-structures: micro-creators prefer product-seeding-only + free-product / mid-tier-creators prefer mid-tier-CPS 20-30%-of-GMV + $100-base-fee / macro-creators prefer macro-flat-fee $500-$5k/creator/campaign OR enterprise-hybrid 5%-of-GMV + $1k-base-fee + content-licensing + whitelisting.

4. **Skipping content-licensing-template before creator-onboarding.** The canonical anti-pattern: brand signs 50+ creators without content-licensing-template, then creator-content lives only on creator's-profile; brand cannot re-promote creator-content as paid-ads or organic-amplification; loses 2-4× creator-content-ROI vs content-licensing-enabled brands per Awin 2024 + Impact 2024 benchmarks. **Fix:** Draft the canonical 3-clause content-licensing-template [perpetual-organic-usage + 90-day-paid-amplification + creator-attribution-required] with legal counsel BEFORE Phase 1 creator-onboarding; require every creator-sign-on to execute content-licensing-agreement.

**C. Content-licensing + whitelisting pitfalls**

5. **Skipping whitelisting-ads-template.** The canonical anti-pattern: brand has creator-content + paid-amplification-budget but no whitelisting-rights; creator-content sits on creator's-profile with no paid-amplification-leverage; loses 2-4× creator-content-ROI vs whitelisting-enabled brands. **Fix:** Draft the canonical whitelisting-ads-template [creator-grants-brand-permission-to-run-creator-content-as-brand-paid-ads] with legal counsel BEFORE Phase 1 creator-onboarding; require every creator-sign-on to execute whitelisting-agreement.

6. **Under-investing in paid-amplification-budget for top-performing-creator-content.** The canonical anti-pattern: brand allocates 10-20% of paid-amplification-budget to creator-content vs 80-90% to brand-original-content; achieves 0.5-1× creator-content-ROAS vs 2-4× creator-content-ROAS for brands allocating 50-70% of paid-amplification-budget to creator-content per Move #15.x Pillar 3 Spark-Ads-boost pattern. **Fix:** Allocate 50-70% of paid-amplification-budget to top-performing-creator-content per Move #15.x Pillar 3; identify top-performers via Triple-Whale-creator-cohort-overlay.

**D. Creator-tier-promotion + content-licensing-renewal pitfalls**

7. **Skipping the creator-tier-promotion-SOP.** The canonical anti-pattern: brand treats all creators as interchangeable; caps at 0.5-1× creator-economy-ROI vs 2-4× for brands with creator-tier-promotion-SOP that promotes top-performing-micro-creators to mid-tier-status + top-performing-mid-tier-creators to macro-status per Move-#15-affiliate-program-tier-promotion-pattern. **Fix:** Build the canonical 4-trigger-tier-promotion-SOP [volume + cohort-LTV + content-quality + tenure]; promote-creator-or-sunset-creator based on 90-day-cohort-LTV-data; quarterly-promotion-cadence-steady-state per Phase 4.

8. **Skipping the 90-day content-licensing-renewal-cadence.** The canonical anti-pattern: brand renews all content-licensing regardless of content-piece-level-ROI; pays content-licensing-fee on underperforming-content-pieces; loses 10-30% of creator-economy-program-ROI per Tubular Labs 2024 + Aspire 2024 benchmarks. **Fix:** Build the canonical 90-day-content-licensing-renewal-or-sunset-decision per content-piece-level-ROI-data from Triple-Whale-creator-cohort-overlay (sunset content-licensing for content-pieces with <0.5× average-creator-content-ROI; renew content-licensing for content-pieces with ≥2× average-creator-content-ROI).

**E. Attribution + measurement pitfalls**

9. **Triple-Whale-creator-cohort-overlay-not-wired.** The canonical anti-pattern: brand launches creator-economy-program + content-licensing + whitelisting-ads without Triple-Whale-creator-cohort-overlay-wire; achieves "looks-good-on-creator-platform-dashboard" but creator-driven-cohort-LTV is invisible; cannot iterate creator-discovery-platform + creator-tier-mix + 5-payout-structures + content-licensing-terms + whitelisting-ads-budget-allocation based on actual-cohort-LTV-data. **Fix:** Wire Triple-Whale-creator-cohort-overlay in Phase 1 (before creator-onboarding); use Triple-Whale-creator-cohort-overlay for weekly-5-way-cohort-LTV-comparison per Pillar 5.

10. **Treating creator-economy-program-ROI as a vanity-metric instead of the canonical "great-band-5:1" driver.** The canonical anti-pattern: brand achieves creator-economy-program-ROI 1:1-3:1 and assumes "close-enough-to-5:1-target"; loses 50% of creator-economy-program-compounding-effect per Tubular Labs 2024 + Aspire 2024 benchmarks. **Fix:** Target creator-economy-program-ROI ≥5:1 steady-state per the canonical "great" band (Tubular Labs 2024 + Aspire 2024); iterate weekly via Triple-Whale-creator-cohort-overlay; flag any sub-5:1 weekly-data-point as a P1 iteration-trigger.

**F. Operational cross-cutting pitfalls**

11. **Launching creator-economy-program before Move #6 Triple-Whale-attribution is live.** The canonical anti-pattern: brand launches creator-economy-program + Aspire + GRIN without Triple-Whale-attribution; cannot measure creator-economy-driven-DTC-attribution-overlap (a creator-economy-customer who later buys from organic-DTC-search is mis-attributed to organic-DTC instead of creator-economy-driven-discovery); undercounts creator-economy-driven-attribution by 30-50% per Triple Whale 2024. **Fix:** Gate creator-economy-launch on Move #6 Triple-Whale-attribution-being-live per the canonical 8-prereq-onboarding-pack; verify all 7-gates-pass per Move #6-verification-recipe before Phase 1 onboarding.

12. **Skipping FTC-compliance-disclosure-language on creator-content-briefs.** The canonical anti-pattern: brand creates creator-content-brief-template without "#ad" / "#sponsored" / "#partner" / "#gifted" disclosure-language; receives FTC-violation-notice + $10k+/violation fine per FTC-2023-2024-enforcement. **Fix:** Include the canonical FTC-compliance-disclosure-language in every creator-content-brief-template per the canonical 16-CFR-Part-255-compliance-SOP; verify disclosure appears in every creator-content-post per FTC-2024-disclosure-monitoring-process; quarterly-FTC-audit-SOP per Phase 4 steady-state.

13. **Confusing creator-economy-program with affiliate-program OR TikTok-Shop-creator-affiliate-pool.** The canonical anti-pattern: brand treats creator-economy-program (Aspire + Collabstr + GRIN + CreatorIQ + Tagger for cross-platform-creator-discovery) as interchangeable with affiliate-program (Refersion + Levanta + Impact for publisher-affiliate-relationships) OR TikTok-Shop-creator-affiliate-pool (TikTok-Shop-Creator-Marketplace for in-platform-TikTok-creators); loses creator-discovery-coverage + caps at 5-15% incremental-revenue vs 10-30% for distinct-track-orchestration. **Fix:** Use creator-economy-program for cross-platform-creator-discovery (TikTok + Instagram + YouTube + Substack + podcast + UGC-creators) per Pillar 1; use affiliate-program for publisher-affiliate-relationships (blogs + deal-sites + coupon-sites) per Move #15 Pillar 1; use TikTok-Shop-creator-affiliate-pool for in-platform-TikTok-creators per Move #15.x Pillar 2; all-three-compound for $500k+ GMV brands.

14. **Not accounting for creator-economy-takes + content-licensing + whitelisting-ads in margin-math.** The canonical anti-pattern: brand offers "20%-off-creator-economy-only-promo" assuming gross-margin-absorbsthe-discount; doesn't account for creator-economy-takes 15-30%-CPS-or-flat-fee + content-licensing-fee + whitelisting-ads + Triple-Whale-creator-cohort-overlay + dedicated-creator-economy-manager = 25-50% of GMV cost-stack per Aspire 2024 + Awin 2024. **Fix:** Build the canonical creator-economy-cost-stack-model per Pillar 1 (creator-payouts 15-30% + content-licensing + whitelisting-ads + Triple-Whale + dedicated-manager = 25-50% of GMV); brands with <25% gross-margin-headroom should defer creator-economy OR offer exclusive-creator-economy-only SKUs at higher price-point per the canonical <25%-gross-margin-deferral-gate.

15. **Launching creator-economy-program on luxury-DTC-brand with MAP-policy-violating SKUs.** The canonical anti-pattern: luxury-DTC-brand with MAP-policy-protected-SKUs ($100+ AOV jewelry / premium-skincare / designer-fashion) launches creator-economy-creator-content-with-prominent-discount-codes; violates MAP-policy; loses wholesale-channel-distribution per Faire 2024 + Ankorstore 2024 cross-reference. **Fix:** Use creator-economy-content-licensing-with-MAP-policy-guardrails (no-public-discount-codes + creator-receives-fixed-fee-instead-of-discount-driven-commission) for luxury-DTC-brands; gate creator-economy-launch on MAP-policy-compatibility-audit per the canonical luxury-voice-downgrade-gate.

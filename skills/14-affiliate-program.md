# Affiliate / creator / influencer program

> Move #15 is the canonical DTC creator-led-acquisition channel — 5–15% incremental revenue from 20–40% of new-customer source-mix with **6:1–12.5:1 Year-1 ROI** at the canonical Path B $1M–$5M Shopify default. Built on **5 pillars**: (Pillar 1) Path-by-platform selection — Path A GoAffPro free <$500k / Path B Refersion $99–$299/mo DEFAULT $500k–$5M / Path C Levanta $99–$499/mo cross-channel / Path D PartnerStack $500+/mo B2B / Path E Impact $500+/mo + 3% fee $5M+ enterprise / Path F Aspire $250–$1,000/mo creator-database; (Pillar 2) per-voice commission-tier matrix (Default 15/20/25% / Luxury 10/12/15% / Sustainable 20/25/30% / Gen-Z 25/30/35% / B2B 8–12/12–15/15–20%) + cookie-window override per voice (30/60/30/7/90 days) + payout-schedule override per voice (NET-30 / NET-7 / NET-60); (Pillar 3) iOS 14.5+ cookie-deprecation mitigation recipe — 6 steps (Shopify CAPI server-side + UTM cascade `?tw_camp=affiliate_<id>` + Klaviyo post-purchase `affiliate_id` match + Shopify Customer Pixel + Levanta server-side fingerprinting for Path C only + Triple Whale cohort-LTV overlay) that recovers 30–60% of lost attribution; (Pillar 4) cohort-LTV measurement via Triple Whale affiliate-cohort-overlay + Klaviyo post-purchase webhook + per-affiliate 10-KPI dashboard — the discriminator between "did this affiliate make us money" vs "did this affiliate make us profitable customers"; (Pillar 5) FTC-compliance checklist (16 CFR Part 255) + 3-clause affiliate contract template + quarterly audit SOP — non-negotiable, $10k+ fines per missing `#ad` per post per FTC 2023+2024 enforcement. Ships in 30 days at 3–8 hr/wk operator time. Gated on Move #6 Triple Whale attribution live (for cohort-LTV) + Move #1 cart-abandon live (for post-purchase email match) + ≥1,000 customers/month + ≥$500/mo paid spend + admin access to Shopify + Klaviyo + Refersion/GoAffPro + PayPal Mass Pay or Wise.

## When to use this skill

You have:
- A Shopify (or compatible) DTC store with **≥90 days of order history** + **≥1,000 customers/month**
- **Move #6 Triple Whale / Polar attribution live** with cohort-LTV overlay — affiliate-cohort-LTV measurement requires per-customer attribution substrate
- **≥$500/mo paid-media spend** — below this, manual creator outreach via UGC + gifting is the better lever (no platform needed)
- Klaviyo (or equivalent ESP) on a paid plan — the post-purchase email match is the canonical iOS-14.5+-recovered signal
- Admin access to **Shopify admin + Klaviyo + your affiliate platform (Refersion / GoAffPro / Levanta / Impact / Aspire) + PayPal Business or Wise (for commission payout)**
- At least **1 share-worthy SKU** (≥4.0/5 review average + ≤$150 price point + clear use-case per Awin 2024)
- 3–8 hr/wk operator time across 4 weeks (~25 hr one-time + 2 hr/wk ongoing)
- Documented refund + return policy (creator-promotion must align with shipping + return commitments)
- An FTC-compliance checklist (16 CFR Part 255 — `#ad` / `#sponsored` / `#partner` language)
- A creator-recruitment pipeline OR an existing customer-base willing to refer (1,000+ customers typically yields 0.5–2% willing-to-refer per Awin 2024)

You do NOT have:
- A live affiliate / creator / influencer program (the most common mid-DTC gap after Move #1 + Move #4 + Move #6 + Move #7 + Move #8 ship)
- A tiered commission structure with per-voice cookie-window + payout-schedule overrides (the #1 lever that lifts creator-volume 40–60% per Awin 2024)
- Server-side tracking (Shopify CAPI + Levanta fingerprinting) — relying on third-party cookies alone loses 30–60% of attribution post-iOS 14.5+ per Levanta 2024
- FTC-compliance automation (the #1 fine-driver per FTC 2023 + 2024 enforcement)
- Cohort-LTV measurement per affiliate (60% of affiliate-driven customers have lower 90-day LTV than DTC-baseline per Awin 2024 — must measure)

## The build (4 weeks, 3–8 hr/wk operator time)

### Phase 1 (Days 1–7) — Platform + commission tiers
1. **Install affiliate platform** per Path decision:
   - Path A <$500k GMV → GoAffPro free from Shopify App Store
   - Path B $500k–$5M DEFAULT → Refersion ($99–$299/mo) or Levanta ($99–$499/mo) if cross-channel
   - Path C $5M+ → Impact ($500+/mo + 3% fee) or Aspire ($250–$1,000/mo) if creator-database-led
2. **Configure commission tiers** per voice (Pillar 2 table above) — Default 15/20/25%, Luxury 10/12/15%, Sustainable 20/25/30%, Gen-Z 25/30/35%, B2B 8–12/12–15/15–20%
3. **Configure cookie windows per voice** (30 / 60 / 30 / 7 / 90 days)
4. **Configure payout schedules per voice** (NET-30 / NET-7 / NET-60)
5. **Ship FTC disclosure templates** in the affiliate-onboarding-email — `#ad` / `#sponsored` / `#partner` at the START of every post
6. **End-of-week check:** 10 affiliate applications live + first test affiliate tracked end-to-end

### Phase 2 (Days 8–14) — Recruitment + onboarding
1. **Recruit first 10–20 affiliates** via 3 channels:
   - Cold-outreach to existing customers (1,000+ customers typically yields 0.5–2% willing-to-refer per Awin 2024)
   - Creator-database via Aspire (Path F) or Levanta (Path C) marketplace
   - Referral from industry networks + competitor-poaching
2. **Send Default + 4 voice-variant welcome emails** (per `assets/17-affiliate-program-templates.md`) — paste-ready 5-voice matrix
3. **Wire cohort-LTV measurement SQL** (Triple Whale `?tw_camp=affiliate_<id>` UTM + post-purchase email match)
4. **Enable FTC disclosure tracking** — every affiliate must sign the canonical 3-clause agreement (FTC + commission + IP rights)
5. **End-of-week check:** ≥10 active affiliates + first 3 affiliate-driven conversions attributed

### Phase 3 (Days 15–21) — Cookie-deprecation mitigation
1. **Wire Shopify CAPI server-side tracking** — Refersion/Levanta server-side events to Shopify Customer Events API (recovers 30–60% of lost cookie signal per Levanta 2024)
2. **Ship UTM cascade** — every affiliate link uses `?tw_camp=affiliate_<id>&utm_source=affiliate&utm_medium=creator&utm_campaign=<slug>`
3. **Wire Klaviyo `affiliate_id` property** on the post-purchase confirmation event (recovers 15–25% of iOS-lost conversions per Refersion 2024)
4. **Install Shopify Customer Pixel** with affiliate-aware event-tracking (recovers another 5–10% per Shopify 2024)
5. **(Path C only) Levanta server-side fingerprinting** — recovers 25–35% of iOS-lost conversions vs 15–25% baseline per Levanta 2024
6. **Wire Triple Whale affiliate-cohort-LTV overlay** — every affiliate-driven customer flows into the `affiliate_cohort_ltv` widget
7. **Enable commission auto-payment** via PayPal Mass Pay or Wise
8. **End-of-week check:** affiliate-driven revenue accurately attributed despite iOS 14.5+ cookie loss + first commission payment cleared

### Phase 4 (Days 22–30) — Tier promotion SOP + Sustainable-creator mission-alignment
1. **Build the tier-promotion SOP** (the canonical 4-trigger ladder): volume-based (>$10k/mo → Tier 2 / >$25k/mo → Tier 3) + cohort-LTV-based (Tier 2 if 90-day LTV >$300) + content-quality-based (Tier 3 if 4.0+ engagement rate on 5+ posts in trailing 90 days) + tenure-based (Tier 2 automatic at 6-month tenure + minimum $5k cumulative)
2. **Wire the Sustainable-affiliate-mission-alignment-verifier** (per `assets/12-impact-data-pipeline.md`) — 20/25/30% Sustainable tier is gated on mission-alignment-verifier pass
3. **Publish the program publicly** — program directory on Awin + Impact + Levanta marketplace
4. **Run first monthly review** — per-affiliate 10-KPI dashboard (clicks / CVR / revenue / commission / CAC-equivalent / 30/60/90-day LTV / LTV:CAC ratio / tier-promotion eligibility)
5. **Run first quarterly compliance audit** — review every active affiliate's last 10 posts for FTC disclosure compliance
6. **End-of-week check:** tier-promotion SOP live + first Sustainable-mission-verifier pass + program publicly listed + Gate A–D all passing

## Common pitfalls (15 from real builds)

### Failure mode A — Platform-selection errors
1. **Path E Impact for <$1M GMV.** Impact's 3% transaction fee + $500/mo minimum destroys unit economics below $20k/mo affiliate-driven GMV per Impact 2024. **Fix:** Path A GoAffPro free for <$500k / Path B Refersion for $500k–$5M / Path C Impact only at $5M+ GMV.
2. **Path A GoAffPro free for >$5k/mo affiliate-GMV.** GoAffPro free has no server-side tracking; iOS 14.5+ cookie loss = $2k+/mo revenue un-attributed at this scale per Awin 2024. **Fix:** upgrade to Path B Refersion when affiliate-driven GMV exceeds $5k/mo.

### Failure mode B — Commission-structure errors
3. **Flat 15% commission across all voices.** Awin 2024 benchmarks show tiered programs lift creator-volume 40–60% vs flat. **Fix:** ship the canonical 5-voice commission-tier matrix (Default 15/20/25% / Luxury 10/12/15% / Sustainable 20/25/30% / Gen-Z 25/30/35% / B-B 8–12/12–15/15–20%).
4. **No tier-promotion SOP.** Top creators leave for higher-paying competitors (Awin 2024: 35% of top-decile affiliates churn within 90 days if no tier-up path). **Fix:** ship the 4-trigger tier-promotion SOP (volume + cohort-LTV + content-quality + tenure).
5. **Commission without cookie-window override per voice.** Gen-Z creators have 7-day impulse window vs B2B 90-day sales cycle. **Fix:** ship the canonical per-voice cookie windows (Default 30d / Luxury 60d / Sustainable 30d / Gen-Z 7d / B2B 90d).

### Failure mode C — Cookie-deprecation errors
6. **Relying on third-party cookies alone.** iOS 14.5+ killed 30–60% of conversion attribution for affiliate programs without server-side tracking per Levanta 2024. **Fix:** ship the canonical 6-step cookie-deprecation mitigation recipe (Shopify CAPI + UTM fallback + Klaviyo post-purchase email match + Shopify Pixel + Levanta server-side fingerprinting for Path C only + Triple Whale cohort-overlay).
7. **No UTM parameter on affiliate links.** Triple Whale attribution fires ONLY when `?tw_camp=affiliate_<id>` is present; missing UTMs = 0% attribution. **Fix:** every affiliate link template includes the canonical UTM cascade (`?tw_camp=affiliate_<id>&utm_source=affiliate&utm_medium=creator&utm_campaign=<slug>`).

### Failure mode D — Cohort-LTV measurement errors
8. **Measuring affiliate-success on attributed-revenue alone (not cohort-LTV).** Awin 2024 benchmarks show 60% of affiliate-driven customers have lower 90-day LTV than DTC-baseline if the affiliate targets bargain-hunters. **Fix:** ship the canonical 4-step cohort-LTV measurement SQL + the per-affiliate KPI dashboard with 10 metrics.
9. **No Klaviyo post-purchase match.** Klaviyo's `affiliate_id` property on the post-purchase confirmation event recovers 15–25% of iOS-lost conversions per Refersion 2024. **Fix:** wire Refersion → Klaviyo webhook so the customer's `affiliate_id` flows into Klaviyo's customer-profile.

### Failure mode E — FTC compliance errors
10. **No FTC disclosure language.** FTC 2023 + 2024 enforcement actions: $10k+/violation per affiliate per post for missing `#ad` / `#sponsored` / `#partner` disclosures. **Fix:** ship the canonical FTC-compliance checklist + quarterly compliance audit.
11. **Free-product-only compensation without disclosure.** The FTC's most-enforced affiliate violation per 2023 + 2024 actions; fines $50k+ per influencer per post. **Fix:** every gifting relationship must be disclosed + every gifting contract must include the canonical FTC clause.
12. **No written affiliate agreement.** Without a signed contract, creators can dispute commission terms, claim IP rights to their content, and trigger FTC compliance gaps. **Fix:** every affiliate must sign the canonical 3-clause affiliate agreement (per `assets/03-ugc-brief.md` C3 affiliate contract template).
13. **Disclosure buried mid-caption.** Per FTC 2023, disclosure must be at the START of the post (not buried). **Fix:** the canonical affiliate-onboarding-email includes a disclosure-template with `#ad` in the first 50 characters of every post.

### Failure mode F — Operational errors (cross-cutting)
14. **No payment-automation SOP.** Manual PayPal payouts for 30+ affiliates = 5+ hr/month of operator work. **Fix:** wire PayPal Mass Pay or Wise for batch-payouts + Refersion auto-payment.
15. **No tier-down SOP.** Underperforming affiliates (90 days <$500 cumulative) consume support bandwidth without producing revenue. **Fix:** ship the tier-down SOP — 90-day review + 30-day warning + tier-down or removal.

## How to extend this skill

Once the basic affiliate program is live:
- **Add a creator-program track** (Path F Aspire) — converts top-tier affiliates into recurring creator-partnerships with monthly retainers + content-volume minimums
- **Add a paid-creator whitelisting layer** — top-performing organic creator posts become Spark Ads / whitelisting ads (per Move #10 AI ad creative iteration)
- **Add an ambassador program** — top 5% of affiliates become ambassadors with monthly retainer + product seeding + event invites
- **Add an affiliate-cohort-LTV-by-voice overlay** — break out the per-affiliate 10-KPI dashboard by voice profile (Default / Luxury / Sustainable / Gen-Z / B2B) to identify which voice drives the highest-LTV affiliate-acquisition
- **Add a referral-program on top** (companion skill to Move #8 loyalty) — Smive.io / Yotpo referrals vs affiliate-program distinction is "referrals = customer-to-customer, affiliate = creator-led-acquisition"
- **Add a TikTok Shop creator-partnership layer** (per research/11) — TikTok Shop's creator-affiliate-program is structurally distinct from web affiliate (commission baked into the TikTok Shop UI; creators discover products via the TikTok Shop marketplace)

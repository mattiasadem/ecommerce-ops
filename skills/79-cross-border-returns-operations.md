## Common pitfalls (18 from real builds)

**1. Treating cross-border returns as a "carrier problem" instead of a multi-jurisdiction legal+logistics+tax+compliance problem.** The merchant's cross-border return-rate climbs to 20-25% but they only wire a per-country carrier-routing table and never build the per-jurisdiction return-eligibility matrix, the per-jurisdiction compliance library, the per-country VAT/GST refund pipeline, the per-country AES/EEI export-control filing, the per-country duty drawback, or the per-country recommerce-handoff. They end up paying 2-4x more in unnecessary carrier-cost + customs-broker-fee + VAT/GST-forfeiture + duty-forfeiture than they should. **Fix:** build all 5 pillars (P1 return-eligibility, P2 carrier-routing, P3 export-control + VAT/GST refund + payment-refund, P4 recommerce-handoff + regulatory re-import compliance, P5 P&L separation + cohort-LTV-overlay + scorecard) at the same time, with explicit per-jurisdiction ownership + per-jurisdiction SLA + per-jurisdiction monitoring.

**2. Using "30-day" as the default return window across all jurisdictions without checking the per-jurisdiction legal minimum.** California SB-707 mandates a 30-day return window for online retailers, EU CRD mandates a 14-day withdrawal window (extendable to 30+ days per Member State), UK CRA mandates a 30-day right-to-reject for faulty goods, Mexico PROFECO mandates a 5-day window for online purchases, Japan SCTA mandates an 8-day window. The merchant picks 30 days globally, ships to Mexico, and gets a PROFECO fine. **Fix:** build the per-jurisdiction withdrawal-window matrix with explicit per-jurisdiction compliance language and wire it to the per-jurisdiction compliance library.

**3. Not localizing the returns-portal into the destination-country language.** The merchant ships to Germany but the returns-portal is English-only with EUR pricing. 30% of German customers abandon the return because they can't read the withdrawal form. **Fix:** localize the returns-portal into the 10 most-common destination-country languages (English / Spanish / French / German / Italian / Portuguese / Dutch / Japanese / Korean / Mandarin) with per-jurisdiction withdrawal-form template + per-jurisdiction restocking-fee-disclosure template.

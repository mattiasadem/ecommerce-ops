## Common pitfalls (18 from real builds)

1. **Wiring an autonomous agent without per-agent-guardrail.** The agent takes a high-stakes action (refund-issued / order-edited / subscription-paused / ad-paused) without a prompt-injection-defense / jailbreak-test / output-redaction / Move #33 fraud-chargeback-defense. **Fix:** wire per-agent-guardrail (Lakera Guard / Prompt Armor / NeMo Guardrails / Azure AI Content Safety / Rebuff / Vigil) FIRST, then wire the agent. Drift fails the canonical agentic-security posture.

2. **Wiring an autonomous agent without per-agent-eval pipeline.** The agent silently drifts in precision/recall and erodes per-agent-CSAT and per-agent-cost-per-resolution. **Fix:** wire per-agent-eval pipeline (Patronus / DeepEval / Ragas / Braintrust / LangSmith Evals / Arize Phoenix / Maxim) FIRST, then wire the agent. Drift fails the canonical agentic-eval posture.

3. **Wiring an autonomous agent without Move #11 AI-CS substrate.** The agent receives a customer-question, but there's no documented Move #11 AI-CS layer to provide the canonical chat-deflection fallback. **Fix:** wire Move #11 AI-CS automation (per skill 11) FIRST, then wire the agent. Drift fails the canonical agentic-customer-service posture.

4. **Wiring an autonomous agent without Move #34 CX-platform substrate.** The agent takes a customer-service action (refund / exchange / escalation) but the CX-platform doesn't surface the cross-channel-history (email → chat → voice → SMS → WhatsApp → Messenger → Instagram DM → TikTok DM) to the agent. **Fix:** wire Move #34 CX-platform (per skill 34) FIRST, then wire the agent. Drift fails the canonical cross-channel-context-relay.

5. **Wiring an autonomous agent without Move #71 order-management substrate.** The agent takes a fulfillment action (reorder / route / cancel) but the OM-orchestration API doesn't expose the canonical inventory-state / 3PL-state / carrier-state to the agent. **Fix:** wire Move #71 order-management (per skill 71) FIRST, then wire the agent. Drift fails the canonical agentic-fulfillment posture.

6. **Wiring an autonomous agent without Move #28 returns-portal substrate.** The agent issues a return-label but the returns-portal doesn't surface the canonical disposition-R1 / R2 / R3 / R4 / R5 per Move #75 recommerce. **Fix:** wire Move #28 returns-portal-orchestration (per skill 28) FIRST, then wire the agent. Drift fails the canonical agentic-returns posture.

7. **Wiring an autonomous agent without Move #33 fraud-chargeback-defense.** The agent issues a refund to a fraudulent customer. **Fix:** wire Move #33 fraud-chargeback (per skill 33) FIRST, then wire the agent. The agent must verify identity + check fraud-signals + check chargeback-history + check return-abuse-rate + check Move #58 brand-protection-counterfeit-enforcement + check Move #69 product-safety-compliance-recall BEFORE issuing the refund. Drift triggers chargeback-loss.

8. **Wiring an autonomous agent without Move #32 sales-tax-VAT substrate.** The agent processes a cross-border refund but doesn't refund the sales-tax correctly. **Fix:** wire Move #32 sales-tax-VAT (per skill 32) FIRST, then wire the agent. Drift fails the canonical sales-tax-compliance posture.

9. **Wiring an autonomous agent without per-agent-token-budget.** The agent loops on a task and burns $50–$500 in token-cost. **Fix:** wire per-agent-token-budget (per-day + per-task + per-action) + per-agent-circuit-breaker (max-iterations + max-tokens + max-actions) FIRST, then wire the agent. Drift fails the canonical agentic-cost-control.

10. **Wiring an autonomous agent without per-agent-model-routing-rule.** Every agent-action runs Sonnet 4.5 even when Haiku 4.5 or Llama 3.3 70B would suffice. **Fix:** wire per-agent-model-routing-rule (high-stakes → Sonnet 4.5 / medium-stakes → Haiku 4.5 / low-stakes → Llama 3.3 70B / Mixtral 8x22B / Gemini 2.0 Flash / DeepSeek V3 / Qwen 2.5) with documented cost-routing. Drift fails the canonical agentic-cost-control.

11. **Wiring an autonomous agent without per-agent-cohort-LTV-overlay.** Every agent-resolved customer is attributed as a generic visitor and the recovered-revenue is invisible. **Fix:** wire per-agent-cohort-tag (agentic_resolved / agentic_deflected / agentic_handoff / agentic_returns / agentic_fulfillment / agentic_ads / agentic_creative / agentic_email / agentic_sms / agentic_csat / agentic_nps / agentic_churn_saved / agentic_winback / agentic_subscription) via Triple Whale / Polar / Northbeam cohort-LTV-overlay.

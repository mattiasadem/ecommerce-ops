## The build (6 phases over 8–16 weeks for a $500k-$5M GMV DTC brand)

### Phase 1 — Org design + role rubrics + headcount plan (Week 1–2)

1. **Map current org chart.** Lay out cofounders + FTE + contractors + part-time + VAs in a single org chart (Google Slides + Lattice or Rippling or BambooHR free-tier org chart).
2. **Define role rubrics for each role.** Use Pave + Radford + Mercer + Culbertson + Option Impact leveling-rubric (IC1-IC7 + M1-M5 + D1-D4 + VP1-VP3 + C-suite), document per-role responsibilities + required-skills + preferred-skills + experience-years.
3. **Build 12-month headcount plan.** Lay out which roles need to be filled in Q1 + Q2 + Q3 + Q4 by function (customer-service + operations + marketing + finance + product + creative + tech + growth), by level (IC vs manager vs director), by full-time vs part-time vs contractor.
4. **Document hiring-budget.** Calculate per-role fully-loaded-cost (base + benefits + payroll-tax + equipment + software + learning-stipend + home-office-stipend).
5. **Document comp-band-rubric.** For each role + level, define min + mid + max base-salary + equity-grant + bonus-target per Pave + Radford + Mercer benchmarks.

### Phase 2 — HRIS + payroll + benefits + equipment (Week 3–5)

1. **Pick HRIS + payroll.** Rippling $8-$22/employee/mo Path B DEFAULT OR Gusto $40-$80/mo Path A SMB OR BambooHR $6-$22/employee/mo Path A SMB OR Justworks PEO $49-$149/employee/mo OR ADP Workforce Now enterprise OR Paylocity enterprise OR Paycom enterprise.
2. **Set up 50-state payroll + tax-nexus.** Wire federal-tax + state-tax + local-tax + unemployment-insurance + workers-comp + disability.
3. **Set up W-2 + 1099-NEC + contractor flows.** Wire W-2-full-time + W-2-part-time + 1099-NEC-contractor with $600+ threshold-tracking.
4. **Set up benefits administration.** Health-insurance + dental + vision + life + disability + FSA + HSA + commuter.
5. **Set up 401k.** Traditional + Roth + safe-harbor + employer-match + vesting-schedule (immediate-vesting or 3-year-graded or 6-year-graded).
6. **Set up equipment + device-management.** Laptop + monitor + keyboard + mouse + headphones + home-office-stipend per Rippling + Apple Business Manager + Windows Autopilot.
7. **Set up time-tracking + PTO + sick-days + parental-leave.** Wire Rippling + Gusto + BambooHR time-tracking + PTO-policy (unlimited-PTO or accrued-PTO or 4-day-workweek).

### Phase 3 — Hiring pipeline + ATS + structured-interview (Week 4–7)

1. **Pick ATS.** Greenhouse $6k-$25k/yr OR Lever $6k-$25k/yr OR Workable $169/mo OR Ashby $250-$1k/user/mo Path B DEFAULT for scaling brands.
2. **Set up structured-interview-rubrics.** Per role, document 4-6 interview-stages + per-stage scorecard + per-stage question-bank + per-stage competency-evaluation.
3. **Wire candidate-sourcing.** LinkedIn Recruiter + LinkedIn Recruiter Lite + LinkedIn Talent Insights + GitHub Recruiter + AngelList + Himalayas + Remote OK + We Work Remotely + YC Work at a Startup + Wellfound + Built In.
4. **Wire job-board distribution.** Indeed + Zip Recruiter + Snag + Monster + Dice + Glassdoor + Simply Hired + Google for Jobs.
5. **Wire candidate-experience.** Career-site + ATS-application-flow + automated-acknowledgement + status-page + rejection-letter-template + candidate-NPS-survey.
6. **Wire structured-interview-loop.** Per stage: 30-60-min interview with 4-6 questions + scorecard + competency-rating + recommendation (strong-hire / hire / no-hire / strong-no-hire).
7. **Wire diversity + inclusion reporting.** Greenhouse DEI-reporting + Lever DEI-reporting + Workable DEI-reporting + Ashby DEI-reporting.

### Phase 4 — Onboarding + week-1 + 30/60/90 review (Week 6–8)

1. **Pick onboarding-platform.** BambooHR Onboarding + Rippling Onboarding + Gusto Onboarding free with HRIS tier.
2. **Document week-1 onboarding-plan.** Day-1: orientation + computer-setup + SSO + Slack + email + benefits-enrollment + first-meeting. Day-2-3: role-specific shadowing + 1:1 with manager + read-the-codebase or read-the-playbook. Day-4-5: first-small-task + buddy-introduction.
3. **Document 30/60/90 review-rubric.** 30-day: foundational-skills + cultural-fit + workflow-ramp. 60-day: first-major-deliverable + first-cross-functional-collaboration. 90-day: full-ramp + first-major-project + first-quarterly-OKR.
4. **Wire equipment-shipment-tracking.** Laptop-shipped + monitor-shipped + peripherals-shipped + home-office-stipend-shipped before Day-1.
5. **Wire first-day-orientation.** Founder-1:1 + culture-deck-walkthrough + benefits-walkthrough + payroll-walkthrough + 401k-enrollment + IT-setup-walkthrough.
6. **Wire first-90-day-metrics.** Time-to-first-deliverable + first-90-day-engagement-survey + first-90-day-1:1-cadence + first-90-day-OKR-completion-rate.

### Phase 5 — Performance management + OKRs + 1:1 + 360 review (Week 8–12)

1. **Pick performance-management-platform.** Lattice $15-$30/user/mo Path B DEFAULT OR Culture Amp $4-$8/employee/mo OR 15Five $4-$8/user/mo OR Officevibe $3-$5/user/mo OR Peakon $3-$5/employee/mo OR Glint $5-$8/employee/mo.
2. **Set up OKR-cascade.** Company-OKRs → Department-OKRs → Team-OKRs → Individual-OKRs with quarterly-cadence + weekly-check-ins.
3. **Set up 1:1 cadence.** Manager-1:1 every 1-2 weeks with structured-template (wins + blockers + feedback + goals).
4. **Set up quarterly-review.** 360-feedback + self-assessment + manager-assessment + skip-level-assessment.
5. **Set up annual-review + comp-refresh.** Annual-review-cycle + comp-refresh-cycle (Q1) + promotion-cycle (Q1 + Q3) + bonus-cycle (Q1 + Q4) + equity-refresh (Q1 + after-fundraise).
6. **Wire pulse-survey + engagement-survey.** Weekly-pulse + quarterly-engagement + annual-engagement + manager-effectiveness-survey.

### Phase 6 — Comp design + equity-admin + benefits expansion + culture + remote-work (Week 12–16)

1. **Pick comp-band-platform.** Pave $20k-$60k/yr OR Ashby Comp OR custom-spreadsheet (NOT recommended for >$5M GMV).
2. **Pick equity-admin-platform.** Carta $500-$2k/yr Path A SMB OR Pulley $500-$2k/yr Path A SMB OR Shareworks enterprise.
3. **Set up equity-grant-policy.** ISO-options + NSO-options + RSU-grants per role + level + vesting-schedule (4yr/1yr-cliff standard) + refresher-grant-policy (annual + after-promotion + after-fundraise).

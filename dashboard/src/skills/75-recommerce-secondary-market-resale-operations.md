# Recommerce + reverse-logistics recommerce + secondary-market resale operations (Move #75)

> Run one grading-to-resale engine that lands every returned item in the right disposition lane (R1 restock / R2 open-box / R3 refurb / liquidate / recycle / donate) at the right cost-per-unit, captures the 25–55% recovered-revenue on R2/R3 returns, separates the recommerce P&L from the primary-channel P&L, sells through the right resale channel (in-house recommerce storefront / B-Stock / Refurbed / Trove / Recurate / Loop Recommerce / ReturnGO Refurb / ThredUp / The RealReal / Vestiaire / Rebag / StockX / GOAT / Back Market / eBay Refurbished / Amazon Renewed / Walmart Restored / Apple Refurbished / Samsung Renewed / Best Buy Outlet / Dell Refurbished / HP Renewed / Sony Refurbished), wires trade-in / take-back / buy-back / store-credit-trade-in loops, attributes the recovered revenue back to the original cohort, and proves it all back to finance, sustainability/ESG, and customers via a monthly recommerce scorecard and a Triple Whale cohort-LTV overlay.

## When to use this skill

Use this skill the first time any of these is true:

- the merchant has ≥2% return-rate on the primary channel and ≥3% of returned units end up in liquidate / donate / landfill even though they are still sellable as open-box or refurbished;
- the merchant has launched or is considering launching a "trade-in" program, a "take-back" program, a "buy-back" program, a "circular" program, a "rental" program, a "subscription-cycle" program, or any other post-purchase reuse program but has no canonical disposition-ladder (R1 → R2 → R3 → liquidate / recycle / donate) to assign returned / trade-in / take-back / buy-back units to;
- the merchant has launched or is considering launching a "secondary market" storefront (e.g., a Shopify "Outlet" or "Restored" collection, a "Re:commerce" subdomain, a B2B outlet) but has no documented grading standard, no per-SKU R1/R2/R3 routing rule, no in-house-refurb-vs-vendor-out decision tree, no per-channel resale cost-per-unit, and no Triple Whale cohort overlay that tags recommerce buyers separately from primary-channel buyers;

### Entry evidence

Do not open a recommerce lane until the minimum evidence is complete:

- SKU list with cost, MSRP, return-rate, prior-condition-flag, refurb-cost ceiling, trade-in eligibility, R1/R2/R3 disposition routing;
- returns portal / carrier-mix / warehouse-management integration (per Move #28, #59, #45) producing a per-return disposition candidate within 24 hours of receipt;
- documented grading standard — R1 (full retail, factory-sealed) / R2 (open-box, fully functional, may have minor cosmetic damage) / R3 (refurb, fully functional, may have visible cosmetic damage) / R4 (salvage / parts only) / R5 (recycle / donate) — with per-tier test / photos / weights / serial-number verification / prior-condition-flagging protocol;
- documented data-erasure compliance for electronics (NIST 800-88 Purge / NIST 800-88 Clear / DoD 5220.22-M / Blancco / WhiteCanyon) with per-unit certificate-of-destruction archived;
- documented resale channel partner (B-Stock / Refurbed / Loop Recommerce / ReturnGO Refurb / Trove / Archive / Recurate / in-house recommerce storefront / Amazon Renewed / eBay Refurbished / Walmart Restored / Apple Refurbished / Samsung Renewed / Best Buy Outlet / Dell Refurbished / HP Renewed / Sony Refurbished / Target Pre-Owned / Back Market / StockX / GOAT / Vestiaire / Rebag / The RealReal / ThredUp) with per-channel commission, per-channel recovered-revenue floor, per-channel warranty-defect-rate SLA, per-channel warranty-disclaimer (FTC Magnusson-Moss / EU Consumer Rights Directive / UK Consumer Rights Act), and per-channel ESG-disclosure-overlay (EU ESPR Digital Product Passport / France AGEC index display / California SB-54 EPR / California Prop 65 warning / EU WEEE / EU Battery Regulation);

### Operator anti-targets

Do not use this skill if any of these is true:

- the merchant has <2% return-rate and <$200k/yr in liquidate / donate / recycle / landfill value — recommerce lanes don't pay back at <$200k/yr;
- the merchant has no documented grading standard, no per-unit disposition routing, no per-channel resale-cost floor, and no Triple Whale / Polar / Northbeam attribution layer;
- the merchant wants a recommerce program but is unwilling to invest the 4–8 hr/wk operator time, the 5–10 hr/wk refurb / grading / refurb-triage time, and the 8–15 hr/wk customer-experience / CX tier-2 / rma / refurb-status / take-back-status time that a real recommerce program requires;
- the merchant is selling high-touch / made-to-order / personalized / perishable / hazmat / regulated goods where recommerce grading is impractical (per Move #69 product-safety-compliance-recall-operations, per Move #72 hazardous-materials-lithium-battery-dangerous-goods);
- the merchant is selling counterfeit-prone / high-counterfeit-risk categories (per Move #58 brand-protection-counterfeit-enforcement) and has no authentication-orchestration capability — recommerce without authentication in counterfeit-prone categories creates a parallel counterfeit-channel risk.

## What "best in class" looks like

Best-in-class recommerce programs have:

- **Single canonical grading engine.** R1 / R2 / R3 / R4 / R5 disposition grading is the same across warehouse, returns portal, refurb partner, in-house refurb, third-party take-back partner, B2B outlet (B-Stock), and resale marketplace (Amazon Renewed / eBay Refurbished / Walmart Restored). Drift between any two surfaces fails CI.
- **Per-unit disposition routing.** Every returned / trade-in / take-back / buy-back / refurb unit is routed within 24 hours of receipt to the optimal disposition lane (R1 restock / R2 open-box / R3 refurb / R4 salvage / R5 liquidate / recycle / donate) based on grading, refurb cost ceiling, resale channel commission, recovered-revenue floor, and per-cohort recommerce-attribution target.
- **Per-channel recovered-revenue floor.** Every resale channel (in-house recommerce / B-Stock / Refurbed / Loop Recommerce / ReturnGO Refurb / Trove / Recurate / ThredUp / The RealReal / Vestiaire / Rebag / StockX / GOAT / Back Market / Amazon Renewed / eBay Refurbished / Walmart Restored / Apple Refurbished / Samsung Renewed / Best Buy Outlet / Dell Refurbished / HP Renewed / Sony Refurbished / Target Pre-Owned) has a documented recovered-revenue-floor (% of MSRP) that triggers automatic disposition routing — units that recover <30% of MSRP on Amazon Renewed route to Refurbed; units that recover <20% route to B-Stock; units that recover <10% route to liquidate / recycle / donate.
- **Per-SKU refurb cost ceiling.** Every SKU has a documented refurb cost ceiling (typically 20–35% of MSRP). Units whose refurb cost exceeds the ceiling route to liquidate / recycle / donate instead.
- **Per-cohort recommerce-attribution overlay.** Every recommerce sale carries a recommerce-attribution overlay (Triple Whale / Polar / Northbeam cohort tag, Klaviyo / Postscript / Attentive / Iterable / Cordial / Braze / HubSpot / Salesforce Marketing Cloud segment tag) that distinguishes recommerce buyers from primary-channel buyers.
- **Per-tier trade-in pricing card.** Every SKU has a documented trade-in pricing card — new / open-box / refurb / salvage — with per-tier (consumer / VIP / loyalty / B2B) trade-in payout (cash vs store-credit), per-tier refurb-cost-ceiling override, and per-tier eligibility-matrix.
- **P&L separation.** Recommerce P&L is separated from primary-channel P&L per CSRD 2026 / SEC Climate Disclosure 2026 / CDP Disclosure 2026 / GRI Standards 2026 / SASB 2026 / TCFD 2026 / EU CSRD reporting / B-Corp reporting requirements. Recommerce-attachment-rate (% of returned units routed to a resale channel vs liquidate / donate / landfill), recommerce-recovered-revenue (% of MSRP recovered), recommerce-contribution-margin (recovered-revenue minus refurb-cost minus disposition-cost minus resale-commission), and recommerce-ESG-disclosure-overlay (kg-CO2-equivalent-avoided-per-unit, water-liter-equivalent-avoided-per-unit, waste-kg-avoided-per-unit, circular-packaging-reuse-loop-rate) are all surfaced on a monthly recommerce scorecard.
- **Data-erasure compliance.** Electronics carry NIST 800-88 Purge / NIST 800-88 Clear / DoD 5220.22-M / Blancco / WhiteCanyon certificates-of-destruction archived per-unit.
- **Warranty-disclaimer compliance.** Refurb / open-box / trade-in inventory carries FTC Magnusson-Moss / EU Consumer Rights Directive / UK Consumer Rights Act / EU ESPR Digital Product Passport / France AGEC index display warranty-disclaimers.
- **ESG-disclosure overlay.** Every recommerce unit carries per-unit ESG-disclosure data (kg-CO2-equivalent-avoided / water-liter-equivalent-avoided / waste-kg-avoided) that feeds the CSRD / SEC Climate / CDP / GRI / SASB / TCFD / B-Corp / Climate Neutral / 1% for the Planet / EU ESPR Digital Product Passport / EU WEEE / EU Battery Regulation / EU Textiles Strategy / France AGEC / California SB-54 / California Prop 65 reporting layer.

## Common pitfalls (18 from real builds)

1. **Treating returns as "outbound cost center" not "recommerce revenue source."** Returns get liquidated / donated / landfilled within 7 days of receipt because no disposition routing exists, no refurb partner is signed, no resale channel is signed, and no per-unit disposition cost / recovered-revenue / refurb-cost-ceiling / data-erasure-protocol is documented. **Fix:** before opening a recommerce lane, sign at least 1 refurb partner + 1 resale channel + 1 B2B outlet (B-Stock) + document the R1/R2/R3/R4/R5 grading standard + document the per-unit disposition cost / recovered-revenue / refurb-cost-ceiling / data-erasure-protocol. Without this, recommerce is impossible.
2. **No documented R1/R2/R3/R4/R5 grading standard.** Warehouse grader marks a unit "open-box" because "it looks fine" with no documented test, no documented photos, no documented serial-number-verification, no documented prior-condition-flagging. Recommerce inventory quality collapses; warranty-defect-rate spikes to 9%+; resale channel rejects 25%+ of inventory. **Fix:** document the grading standard before opening the lane. Per-tier test + per-tier photos + per-tier serial-number-verification + per-tier prior-condition-flagging. Drift between any two graders fails CI.
3. **No per-SKU refurb cost ceiling.** Refurb partner refurbishes a $50 unit for $40 — refurb cost exceeds the recovered revenue. Margin is negative. **Fix:** every SKU has a documented refurb cost ceiling — typically 20–35% of MSRP for electronics, 8–18% of MSRP for apparel / footwear / soft goods. Units whose refurb cost exceeds the ceiling route to liquidate / recycle / donate instead. The ceiling is enforced by the disposition routing function.
4. **Selling refurb / open-box / trade-in without data-erasure for electronics.** Refurbished laptops with prior-user data, refurbished phones with prior-user accounts, refurbished tablets with prior-user iCloud / Google accounts. Data-leak liability. GDPR / CCPA / PIPEDA / LGPD / FTC Magnusson-Moss / EU Right-to-Repair violations. **Fix:** wire NIST 800-88 Purge / NIST 800-88 Clear / DoD 5220.22-M / Blancco / WhiteCanyon data-erasure for every electronics unit routed to R3 / R4 / R5. Per-unit certificate-of-destruction archived. Per-unit chain-of-custody recorded.
5. **Selling refurb / open-box / trade-in without warranty-disclaimer compliance.** Refurbished laptop sold as new with no warranty-disclaimer; refurbished apparel sold as new with no "open-box / pre-owned / refurbished" disclosure; trade-in phone sold without Prop 65 warning. FTC Magnusson-Moss / EU Consumer Rights Directive / UK Consumer Rights Act / EU ESPR Digital Product Passport / France AGEC index display / California Prop 65 violations. **Fix:** wire per-listing warranty-disclaimer (FTC Magnusson-Moss / EU Consumer Rights Directive / UK Consumer Rights Act / EU ESPR Digital Product Passport / France AGEC index display / California Prop 65) for every refurb / open-box / trade-in listing. Per-listing warranty-disclaimer archived. Per-listing disclosure verified pre-listing.
6. **Refurb-cost-ceiling too low or too high.** Too low → 50%+ of units route to liquidate / donate when they could have been refurbished. Too high → refurb cost exceeds recovered revenue on 25%+ of refurb units → negative margin. **Fix:** calibrate refurb-cost-ceiling per SKU based on actual refurb-cost data from the refurb partner. Review monthly. Recommerce-attribution-overlay cohort LTV should trend positive within 90 days.
7. **Resale-channel commission too high.** Refurb partner takes 35% commission on every refurb sale; recovered-revenue-floor is 35% of MSRP but commission is 35%, leaving zero margin. **Fix:** negotiate resale-channel commission to ≤25% of MSRP for B2B outlet (B-Stock) and ≤20% of MSRP for refurb marketplace (Refurbed / Loop Recommerce / ReturnGO Refurb / Back Market). Document the commission in the partner agreement. Review quarterly.
8. **No documented trade-in pricing card.** Trade-in payout is calculated ad-hoc by the CS agent; payout is inconsistent across customers; payout is non-compliant with FTC Mail-Order Rule / FTC Click-to-Cancel / EU Consumer Rights Directive / California SB-707 / NY SB-4104. **Fix:** document the trade-in pricing card before opening the trade-in lane. Per-tier (consumer / VIP / loyalty / B2B) trade-in payout (cash vs store-credit), per-tier refurb-cost-ceiling override, per-tier eligibility matrix, per-tier payout-velocity (instant-store-credit vs NET-30-cash). Drift between any two CS agents fails CI.
9. **Trade-in store-credit-vs-cash decision is wrong.** Store-credit at 30% of MSRP is more attractive than cash at 20% of MSRP — but the merchant wants to keep cash on hand. Customer chooses cash; merchant loses the customer-LTV halo. **Fix:** store-credit default for consumer / VIP / loyalty tier; cash default for B2B tier; cash optional for consumer tier at 20% of MSRP. Document the decision tree. Wire the Klaviyo / Postscript / Attentive flow to surface the right offer to the right tier.
10. **Trade-in / take-back / buy-back program cannibalizes primary-channel sales.** Customer takes the trade-in payout, buys the new product from a competitor, the merchant loses the primary sale. **Fix:** trade-in payout is store-credit by default (locked to the merchant's catalog). Cash payout requires ≥30-day cooldown + Klaviyo / Postscript / Attentive trade-in-cohort re-engagement flow + Triple Whale cohort-LTV-overlay tracking. Recommerce-attribution-overlay cohort LTV delta should be +12–55% within 90 days.
11. **No Triple Whale cohort overlay tagging recommerce buyers separately from primary-channel buyers.** Recommerce-attribution-overlay is invisible to FP&A. Recommerce revenue is double-counted with primary-channel revenue (refund + resale). CSRD / SEC Climate Disclosure / CDP / GRI / SASB / TCFD reporting is wrong. **Fix:** tag every recommerce sale with `cohort:recommerce_buyer` and every primary-channel sale with `cohort:primary_buyer`. Push trade-in-program participants as `cohort:trade_in_participant`. Push take-back-program participants as `cohort:take_back_participant`. Build the cohort LTV dashboard.
12. **No ESG-disclosure-overlay integration with CSRD / SEC Climate / CDP / GRI / SASB / TCFD.** Per-unit kg-CO2-equivalent-avoided / water-liter-equivalent-avoided / waste-kg-avoided / circular-packaging-reuse-loop-rate is not tracked. CSRD / SEC Climate Disclosure / CDP / GRI / SASB / TCFD reporting is incomplete. EU ESPR Digital Product Passport / EU WEEE / EU Battery Regulation / EU Textiles Strategy / France AGEC / California SB-54 / California Prop 65 reporting is incomplete. **Fix:** wire per-unit ESG-disclosure data into the CSRD / SEC Climate / CDP / GRI / SASB / TCFD / B-Corp / Climate Neutral / 1% for the Planet / EU ESPR / EU WEEE / EU Battery Regulation / EU Textiles Strategy / France AGEC / California SB-54 / California Prop 65 reporting layer.
13. **Recommerce inventory contaminates primary-channel inventory.** R1 restock inventory routes back to primary-channel inventory; an open-box unit ships to a primary-channel customer as new. Customer files chargeback. CSAT drops. Refund rate spikes. **Fix:** recommerce inventory is segregated from primary-channel inventory. Recommerce inventory has its own SKU-suffix (e.g., `-R2`, `-R3`), its own bin-location, its own listing-page, its own warranty-disclaimer, its own attribution-overlay. Cross-contamination is a release-blocking bug.
14. **Resale-channel warranty-defect-rate is unmonitored.** Resale-channel partner returns 12% of inventory as defective; merchant refunds the customer, eats the disposition cost, eats the resale-commission, and writes off the inventory. **Fix:** monitor warranty-defect-rate per resale channel per month. Trigger partner renegotiation if rate exceeds 5%. Trigger channel-routing-shift if rate exceeds 9%. The channel-routing-shift routing function is wired in Day 3 disposition routing.
15. **Refurb partner SLA is unmonitored.** Refurb partner takes 28 days to refurbish a unit; merchant's recommerce-attach-rate drops; resale-channel inventory pipeline starves. **Fix:** monitor refurb-SLA per refurb partner per month. Trigger partner renegotiation if SLA exceeds 7 days for electronics / 14 days for apparel / footwear / soft goods. Trigger partner-diversification (sign a 2nd refurb partner) if SLA exceeds 14 days.
16. **Customer doesn't know the recommerce / trade-in / take-back / buy-back program exists.** Recommerce-attach-rate stays at 4%. Recommerce-recovered-revenue stays at 12% of MSRP. Recommerce-contribution-margin stays negative. **Fix:** wire Klaviyo / Postscript / Attentive / Iterable / Cordial / Braze / HubSpot / Salesforce Marketing Cloud recommerce flow: trade-in offer (Day 14 post-purchase) → take-back offer (Day 60 post-purchase) → refurb-cycle offer (Day 180 post-purchase) → buy-back offer (Day 365 post-purchase).

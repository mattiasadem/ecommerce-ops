# Marketplace expansion

> Move #13 is the canonical second-largest absolute-revenue move in the workspace (behind Move #14 lifecycle-marketing at 22:1 default ROI but ahead of it on dollars-at-stake at $5M+ GMV) — capturing the **30–60% of category demand that starts on Amazon-search-first** (per Jungle Scout 2024: 56% of US product searches start on Amazon vs 15% on Google Shopping vs 8% on direct-DTC site search) via **5 pillars**: (Pillar 1) per-marketplace channel-economics — 1P vs 3P vs hybrid on Amazon (3P DEFAULT $500k–$10M / 1P at $10M+ vendor-manager / hybrid $5M+) + Walmart 3P + Target Plus Roundel 1P (5–10% approval rate) + Amazon EU + bol + Zalando + Cdiscount + Amazon JP; (Pillar 2) brand-canary protection — the **#1 marketplace pitfall is "Amazon Halo" (10–35% of DTC-direct organic-search traffic migrates to Amazon-search-first per Marketplace Pulse 2024)** — mitigated by the canonical 5-lever recipe (Brand Registry + Vine for review-acquisition + Buy-Box ownership >90% + Amazon DSP branded-keyword defense + Amazon Posts + Amazon Transparency); (Pillar 3) operational model — FBA DEFAULT for fast-movers (Prime badge is the single largest Amazon conversion-driver) + FBM for slow-movers + SFP for premium categories + WFS for Walmart + Target Plus for $5M+ category-aligned brands; (Pillar 4) category fit + regulatory friction — high-fit (apparel + home + beauty + pet + jewelry + electronics) / medium-fit (food + supplements + cosmetics — gated + FDA + lot-tracking) / low-fit (CBD + alcohol + medical-devices) / fail (live plants + aerosols + bulk lithium); (Pillar 5) attribution merge — the canonical hard-problem solved by Amazon Attribution + Triple Whale Marketplace Sync + Klaviyo post-purchase `marketplace_purchaser_30d` segment + Walmart Connect + Amazon Customer Engagement. Built on **3 GMV-tier paths** — Path A Amazon-only $500k–$5M (8:1 ROI default) / Path B Amazon + Walmart DEFAULT $1M–$10M (8.3:1 ROI, $2.25M Year-1 incremental net at $5M base) / Path C all marketplaces + international $5M–$50M (10:1 ROI, $9M Year-1 incremental net at $15M base). Ships in 24 weeks across 4 phases (Phase 1 Amazon-only ~25hr / Phase 2 Amazon+Walmart ~20hr / Phase 3 international ~75hr / Phase 4 Target Plus + steady-state ~30hr). Gated on Move #1 + Move #4 + Move #6 + Move #8 shipped + registered USPTO trademark + Amazon Brand Registry approved + ≥$500k US DTC GMV for 3 consecutive months + 5+ hr/wk operator capacity.

## When to use this skill

You have:
- A Shopify (or BigCommerce / Magento / WooCommerce) DTC store with **≥$500k US DTC GMV for 3 consecutive months** — below this the operator is too lean for marketplace-PPC + marketplace-CS + marketplace-returns overhead (defer until $500k threshold)
- **Move #1 cart-abandon + Move #4 welcome + Move #6 Triple Whale / Polar attribution + Move #8 loyalty all shipped** — Move #6 cohort-LTV is required for the canonical cannibalization-measurement recipe (you cannot measure whether marketplace-revenue is incremental vs cannibalized without it)
- A **registered USPTO trademark** (registration certificate in hand, not application receipt — Brand Registry requires 8–12 months for USPTO registration and you cannot ship Phase 1 without Brand Registry approved)
- **At least 1 brand-hero SKU** in a marketplace-fit category (apparel + home + beauty + pet + jewelry + consumer-electronics = high-fit; food + supplements + cosmetics = gated but workable; CBD + alcohol + medical-devices = low-fit; live plants + aerosols + bulk lithium = fail)
- Klaviyo (or equivalent ESP) on a paid plan — the post-purchase-email marketplace-purchaser segment is the canonical iOS-14.5+-recovered attribution signal for marketplace orders
- Admin access to **Amazon Seller Central + Walmart Seller Center + Triple Whale (with Marketplace Sync add-on) + Amazon Attribution dashboard + Shopify admin**
- 5–25 hr/wk operator capacity across 24 weeks (Path A ~25hr one-time + 5hr/wk ongoing / Path B ~45hr + 10hr/wk / Path C ~120hr + 25hr/wk + dedicated marketplace manager)
- Documented FDA-compliance documentation IF category is gated (supplements need Supplement Facts panel + FDA facility registration + lot-tracking; cosmetics need FDA cosmetic-product-registration)
- A fulfillment partner capable of FBA-prep services ($1–$3/unit for first 5 inbound shipments — the operator typically outsources prep to a 3PL)
- An EU compliance partner IF Path C / Phase 3 (CE marking 4–12 weeks + VAT-MOSS 4–8 weeks + EPR per country + GPSR Responsible-Person 1–4 weeks + budget $10k–$30k one-time + $2k–$10k/yr ongoing)

You do NOT have:
- An Amazon Seller Central account OR a Brand-Registry-enrolled brand (the canonical biggest marketplace-expansion gap; most DTC brands have 0% Amazon presence until they ship Move #13)
- Buy-Box ownership >85% on existing Amazon listings (the canonical Buy-Box defense recipe: FBA + competitive-pricing + healthy account health metrics + inventory-availability)
- A Triple Whale Marketplace Sync subscription (the canonical attribution-merge stitch that makes marketplace-revenue measurable against DTC-revenue at the customer-cohort level)
- An Amazon DSP branded-keyword defense budget ($500–$2k/mo to bid on the operator's own brand-name + brand-name + product-name + top-20 SKU-names — without this, resellers and counterfeiters dominate branded search)
- Per-country EU returns-infrastructure (Path C only; without this, returns-cost erodes 5–10% of EU-revenue)
- A documented Amazon Halo measurement protocol (DTC-cannibalization-rate measurement pre-launch vs post-launch — without this, the operator over-credits Amazon with revenue that should be attributed to the brand's broader top-of-funnel presence)

## The build (4 phases over ~24 weeks, 5–25 hr/wk operator time)

### Phase 1 (Weeks 1–4) — Amazon-only

1. **Confirm Brand Registry approved** (USPTO trademark registration certificate + Amazon Brand Registry approval). Without this, you cannot list in gated categories + use A+ Content + run Sponsored Brands + use Amazon Vine + file IP complaints.
2. **Activate Amazon Professional Seller account** ($39/mo). Pick 3P vs 1P per the canonical decision rule: $500k–$10M = 3P + FBA; $10M+ with vendor-manager = 1P; $5M+ with category complexity = hybrid.
3. **Send first hero SKU to FBA** — 8–12 weeks of expected demand. Use Amazon's "Send to Amazon" workflow (generates warehouse-assignment + FNSKU-labeling spec + carton-content-declaration per Amazon's 2024 spec).
4. **Enroll all hero SKUs in Amazon Vine** ($200/SKU + free product). Target: 50+ reviews per hero SKU in first 30 days.
5. **Launch Amazon Sponsored Products auto-campaign** ($2k/mo budget). Set ACoS target <40% by Week 8.
6. **Set up Amazon Posts** (free; image-rich posts in category feeds). Post 4–8 product-images/week per hero SKU.
7. **Set up Amazon DSP branded-keyword defense** ($500/mo bid on own brand-name + brand-name + product-name + top-20 SKU-names).
8. **Wire Amazon Attribution + Triple Whale Marketplace Sync** — link Amazon-ads to Shopify-conversion-tracking + stitch Amazon-orders into Triple Whale's cohort-LTV framework.
9. **End-of-week check:** Brand Registry approved + Buy Box >85% + 50+ reviews per hero SKU within 30 days + ACoS <40% by Week 8.

### Phase 2 (Weeks 5–12) — Amazon optimization + Walmart launch

1. **Launch Amazon Sponsored Brands** (headline-search-ads on operator's brand-name + top-10 product-names). Target ACoS <30%.
2. **Scale Amazon DSP defensive-bidding** to $500–$2k/mo on operator's own branded keywords + retarget shoppers who viewed the product but didn't buy.
3. **Activate Walmart Seller Center + Walmart Brand Portal** (free). Apply for WFS (Walmart Fulfillment Services).
4. **Send first hero SKU to WFS** — 2-day-shipping-capable. WFS inbound 8–12 weeks of expected demand.
5. **Launch Walmart Sponsored Products** ($500/mo budget). Target ACoS <35% + Buy Box >90%.
6. **Apply for Walmart Restored** (optional recommerce / refurbished program).
7. **Wire Walmart Connect attribution** to Triple Whale cohort-LTV overlay.
8. **End-of-week check:** Gate A done-when all 6 criteria met + Walmart Brand Portal approved + 50+ reviews per hero SKU on Walmart within 60 days + Walmart ACoS <35% + Walmart-Buy-Box >90% + DTC-cannibalization-rate <20% by end of Phase 2.

### Phase 3 (Weeks 13–24) — International marketplaces (Path C only)

1. **Register Amazon EU VAT-MOSS** in primary EU country (typically DE for highest sales-volume). 4–8 weeks timeline.
2. **Complete CE marking** for all EU-sold SKUs. 4–12 weeks testing + certification.
3. **Register EPR** in each EU country where products are sold (Germany LUCID + France EPR + Spain + Italy + Netherlands).
4. **Appoint GPSR Responsible-Person** (1–4 weeks service-setup). Required for EU marketplace listings under Dec 2024 EU GPSR.
5. **Activate Amazon EU 5-marketplace seller accounts** (DE + FR + IT + ES + NL). FBA inbound per marketplace (4–8 weeks per marketplace).
6. **Activate bol seller account** (NL + BE).
7. **Apply for Zalando Partner Program** (60% approval rate).
8. **Activate Cdiscount seller account** (FR).
9. **End-of-week check:** Gate C done-when all 8 criteria met + per-marketplace AOV within ±15% of US-baseline AOV + per-marketplace CVR within ±20% of US-baseline CVR by end of Phase 3.

### Phase 4 (Quarter 2+) — Target Plus + steady-state (Path C only)

1. **Apply for Target Plus Roundel** (5–10% approval rate; 8–12 week review). Roundel is Target-handled 1P — selective invite only.
2. **Activate Walmart International** (CA + MX).
3. **Apply for Amazon SFP upgrade** (Seller Fulfilled Prime; 90-day trial + ongoing-metrics bar).
4. **Build cross-marketplace inventory-routing algorithm** — per-marketplace demand-forecast + per-marketplace-warehouse-ship-routing.
5. **Build per-marketplace PPC-budget-allocation algorithm** — per-marketplace ACoS target + per-marketplace CAC payback.
6. **Set up per-marketplace returns-management protocol** — per-marketplace returns-warehouse OR Amazon FBA pan-EU returns ($3–$5/unit).
7. **Hire dedicated marketplace manager** (full-time OR fractional consultant).
8. **Build cross-marketplace-blended-LTV:CAC dashboard** — Triple Whale Marketplace Sync + per-marketplace-attribution-stitched.
9. **End-of-week check:** Gate D done-when all 9 criteria met + cross-marketplace-inventory-turn >6x/yr + per-marketplace CAC payback <12 months + blended-LTV:CAC >3:1 across all channels by end of Year 1.

## Common pitfalls (15 from real marketplace-expansion builds)

### A — Channel-economics errors

**Pitfall #1 — Launching on Amazon without Brand Registry enrolled.** Without Brand Registry, the operator cannot list in gated categories, use A+ Content, run Sponsored Brands, use Amazon Vine, file Brand-Registry IP complaints. Result: low organic rank + no review-acquisition + no defensive-positioning against resellers + no IP-complaint tool to remove counterfeit listings. **Fix:** file USPTO trademark application immediately + ship DTC-only until registration clears (8–12 month USPTO timeline) + enroll in Brand Registry the day registration clears + ship Phase 1 only after Brand Registry is approved.

**Pitfall #2 — Launching on Amazon 3P when 1P is the better model.** 3P vs 1P is not a default decision. 1P (Vendor Central) is invite-only and gives: Amazon-marks-up-on-retail + Amazon-handles-fulfillment + Amazon-handles-CS + Amazon-controls-pricing (NOT the brand). 3P (Seller Central) is open-access and gives: brand-controls-pricing + brand-handles-fulfillment (or uses FBA) + brand-handles-CS (or uses FBA-returns). **Canonical decision rule:** $10M+ brands with vendor-manager relationships → 1P; $500k–$10M brands → 3P + FBA; $5M+ brands with category-management complexity → hybrid (1P for visibility-driving SKUs + 3P for margin-driving SKUs). **Fix:** at $10M+ GMV, apply for Amazon Vendor Central via Amazon's brand-management team; at $500k–$10M, default to 3P + FBA.

**Pitfall #3 — Ignoring marketplace fee-stack math and treating Amazon-revenue as equivalent to DTC-revenue.** Marketplace total cost is **30–45% of Amazon-revenue** (referral 8–15% + FBA pick+pack + FBA storage + Amazon advertising 10–25% + returns + Vine + closing fee) vs DTC total cost **20–30% of Shopify-revenue**. The 10–15% margin compression means Amazon-revenue looks great in absolute terms but is **less profitable per dollar**. **Fix:** compute marketplace-cost as % of Amazon-revenue + marketplace-net-margin as % of Amazon-revenue + compare to DTC-net-margin; if marketplace-net-margin < 50% of DTC-net-margin, marketplace expansion is structurally less profitable than scaling DTC.

### B — Brand-canary-protection errors

**Pitfall #4 — Skipping Amazon Vine for review-acquisition.** Without 50+ reviews in the first 30 days, the listing has no organic rank + no Buy-Box-defense + no shopper-trust. Vine delivers 5–30 verified-purchase reviews within 30 days. **Fix:** enroll all hero SKUs in Amazon Vine at launch ($200/SKU + free product) + budget 6 months of Vine-enrollment per new SKU.

**Pitfall #5 — Losing Buy Box due to FBM + competitive-pricing + unhealthy-account-health.** Buy Box is awarded to the lowest-priced FBA offer with healthy account health metrics. FBM sellers lose Buy Box 10–30% of the time. Operators who drop Buy Box see sales collapse within 24 hours. **Fix:** use FBA for fast-movers (not FBM) + maintain Order Defect Rate <1% + Late Shipment Rate <4% + Pre-fulfillment Cancel Rate <2.5% + maintain inventory-availability (the canonical "out-of-stock loses Buy Box" trap).

**Pitfall #6 — Skipping Amazon DSP branded-keyword defense.** Without Amazon DSP, when a shopper searches "[operator brand name]" on Amazon, the operator's listing shows up but so does every reseller. Structural solution: bid on the operator's own brand-name + brand-name + product-name + top-20 SKU-names via Amazon DSP at $0.50–$2.00 CPC. **Fix:** set up Amazon DSP via Amazon's account-management team + bid $500–$2,000/mo on the operator's own branded keywords + retarget shoppers who viewed the operator's product but didn't buy.

**Pitfall #7 — Amazon Halo not measured.** Per Marketplace Pulse 2024: brands that list on Amazon see 10–35% of their previously-DTC-direct organic-search traffic migrate to Amazon-search-first. Without measuring this, the operator over-credits Amazon with revenue that should be attributed to the brand's broader top-of-funnel presence. **Fix:** measure DTC-cannibalization-rate monthly (Shopify-direct-traffic attributable to brand-name searches, pre-launch vs post-launch) + adjust Amazon-PPC bid strategy if DTC-cannibalization-rate exceeds 20%.

**Pitfall #8 — Skipping Amazon Posts for organic brand-discovery.** Amazon Posts is the Amazon-equivalent of Instagram-feed (free; image-rich posts in category feeds). Without Posts, the brand has no organic-discovery-mechanism within Amazon's browsing-flow (only paid-ads-discovery). **Fix:** set up Amazon Posts via Brand Registry + post 4–8 product-images/week per hero SKU + monitor Posts-engagement via the Amazon Posts dashboard.

### C — Operational-model errors

**Pitfall #9 — Sending slow-movers to FBA.** FBA long-term-storage-fee applies to inventory >180 days at $0.50–$2.50/unit depending on age-bucket. Slow-movers that sit in FBA >6 months erode the FBA cost-advantage. **Fix:** send fast-movers (8–12 weeks of expected demand) to FBA + send slow-movers to FBM or a backup 3PL + monitor Inventory Performance Index (IPI) score monthly + send aged-inventory-removal requests for inventory >180 days.

**Pitfall #10 — FBA inbound-shipping mishaps (wrong warehouse + wrong labeling + wrong packaging).** FBA inbound has strict requirements: shipment to the correct Amazon warehouse (Amazon assigns; sending to wrong warehouse = refusal) + per-unit labeling with FNSKU (not UPC) + polybag-or-box requirements per category + carton-content-declaration per Amazon's 2024 spec. Mishaps cause: inbound-refusal (inventory sent back at operator's expense) + stranded inventory + fee-surcharges for non-compliant packaging. **Fix:** use Amazon's Send to Amazon workflow (the official tool that generates warehouse-assignment + labeling-spec) + use FBA prep services from a 3PL ($1–$3/unit) for first 5 inbound shipments + budget 8 hours of operator-time for first FBA inbound.

**Pitfall #11 — No returns-management recipe.** Marketplace returns are 5–15% of units (vs DTC returns 2–5%). FBA-returns have specific rules: returns to operator's returns-warehouse only on operator-request + Amazon-FBA-returns-storage-fee applies to returns >30 days + Amazon offers "Grade-and-Resell" for returned-inventory (Amazon-refurbishes-and-resells at a discount; operator gets 60–80% of original-Amazon-price back). **Fix:** set up a returns-management protocol: weekly review of FBA-returns + Grade-and-Resell-enrollment for returns-eligible SKUs + returns-storage-fee-budget ($0.50–$2/unit/month) + operator's returns-warehouse for returns-that-Amazon-can't-resell.

### D — Regulatory-friction errors

**Pitfall #12 — Launching on Amazon EU without CE marking + VAT-MOSS + EPR + GPSR compliance.** Without these 4 EU compliance pieces, the operator's Amazon EU listings get suspended within 30–90 days of launch + the operator is liable for per-day-fines for non-compliance. **Fix:** ship all 4 compliance pieces before any Amazon EU launch (CE marking 4–12 weeks + VAT-MOSS registration 4–8 weeks + EPR registration 4–8 weeks + GPSR Responsible-Person appointment 1–4 weeks) + budget $10k–$30k one-time + $2k–$10k/yr ongoing for EU compliance.

**Pitfall #13 — Gated-category launch without FDA / lot-tracking / brand-registry approval.** Gated categories (food + vitamins + supplements + cosmetics + over-the-counter health) require pre-launch approval from Amazon's category-management team + FDA-compliance documentation (Supplement Facts panel + FDA facility registration for supplements + FDA cosmetic-product-registration for cosmetics). Operators who launch without these get listings-suspended within 30 days. **Fix:** file category-gating approval 4–8 weeks before launch + ensure FDA-compliance-documentation is in hand + lot-tracking-system in place (per-unit lot-code + lot-code-on-listing).

**Pitfall #14 — Per-country returns-infrastructure not in place.** Amazon EU requires per-country returns-infrastructure (customer returns to operator's per-country warehouse, not a central warehouse). Without this, returns-cost erodes 5–10% of EU-revenue. **Fix:** set up per-country returns-warehouses via EU-3PLs (SEKO Logistics EU + DHL EU-Returns) OR use Amazon's FBA pan-EU returns (Amazon-handles-EU-returns at $3–$5/unit).

### E — Attribution-measurement errors

**Pitfall #15 — No Triple Whale Marketplace Sync + Amazon Attribution + post-purchase email attribution stitches.** Without these 3 attribution stitches, the operator cannot measure: (a) how much marketplace revenue is incremental vs cannibalized from DTC, (b) what the blended LTV:CAC is across Shopify + Amazon + Walmart, (c) whether the marketplace-expansion investment is paying back at the same rate as the next dollar into Meta-ads or Klaviyo-flows. The structural recipe bias toward "Amazon is great" when in fact Amazon-gain minus DTC-loss is closer to break-even than the operator thinks. **Fix:** ship Triple Whale Marketplace Sync + Amazon Attribution + post-purchase email attribution in Phase 1 + measure DTC-cannibalization rate monthly + compute blended-LTV-by-source-channel quarterly + adjust Amazon-PPC bid strategy based on the blended-LTV-by-source view (NOT on Amazon-attribution-only view).

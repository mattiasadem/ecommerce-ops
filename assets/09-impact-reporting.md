## Goal

Every DTC brand with sustainability commitments needs an **annual + quarterly impact report** that maps what the brand actually did to what the brand said it would do — and the report must be paste-ready for the operator (not a 200-page ESG consultancy deliverable). Without an impact report, the brand's sustainability claims are exactly that — claims, unverifiable by customers, partners, and increasingly by regulators (the EU's Corporate Sustainability Reporting Directive [CSRD] + the US SEC's proposed climate-disclosure rule + California's SB-253 / SB-261 all require verifiable impact reporting for brands above certain revenue thresholds). The Sustainable voice profile (per Asset 02) is the canonical primary voice for impact reporting, but Default / Luxury / Gen-Z / B2B each have a distinct impact-report tone that the operator must apply to avoid greenwashing-trap language.

This impact-report template ships a paste-ready framework that the operator can apply on day 1, even **before they pick data sources** — the 6-pillar framework itself is the durable artifact, and the operator fills in data sources + numbers later:

1. **A 6-pillar impact-reporting framework** — Carbon / Materials / Labor / Packaging / Community / Certification. Each pillar has 5 voice-driven override columns (Default / Luxury / Sustainable / Gen-Z / B2B) + a 1–5 maturity scale + 3–5 concrete sub-questions + a "What moves this pillar" attribution block.
2. **A paste-ready 6-pillar reporting template** — 6 pillar sections × 5 voice-driven override tables = 30 cells of "what the operator's voice changes about how to report this pillar." Sustainable brands report Carbon in tons CO2e shipped + Materials in % recycled + Labor in % living-wage + Packaging in % recycled-input + Community in $ donated + Certification in B-Corp-score. Default brands report the same metrics but in customer-friendly language ("the carbon equivalent of X flights"). Luxury brands report the same metrics but in heritage-language ("the Florence atelier has used solar power since 2018"). Gen-Z brands report the same metrics but in meme-aware-language ("we offset enough carbon to plant Y trees, but the real flex is...").
3. **A data-source wiring recipe** — 6 canonical data sources for the 6 pillars (Shopify Planet / EcoCart / Patch for carbon / Bluesign / Oeko-Tex / GOTS for materials / Fair Trade USA / B Corp for labor / How2Recycle for packaging / 1% for the Planet / B Corp / Climate Neutral Certified for certification) with cost / time-to-data / data-depth / when-to-use columns.
4. **A voice-driven override matrix** — 5 voice profiles × 6 pillars = 30 cells of voice-specific reporting tone. Sustainable brands lead with mission + numbers + certification. Default brands lead with customer-relevant translation ("your last shipment = 2.3 lbs CO2 = a 5-mile drive"). Luxury brands lead with heritage + scarcity + provenance. Gen-Z brands lead with humor + accountability + transparency. B2B brands lead with compliance + ROI + audit-trail.
5. **5 verification gates** — one per non-certification pillar (Gate A Carbon-verification via Shopify Planet API + supplier energy invoices / Gate B Materials-verification via supplier certificates + GOTS / Oeko-Tex chain-of-custody / Gate C Labor-verification via Fair Trade USA audit + supplier wage statements / Gate D Packaging-verification via How2Recycle label audit + supplier FSC certificates / Gate E Community-verification via 1% for the Planet donation receipts + Move #8 loyalty-impact-rewards ledger). The Certification pillar (Gate F) is binary: either the brand has the certification or it doesn't, with a documented audit date.

**Honest-read called out in 6 places:** (a) **the framework ships before data sources are picked** — operators should apply the 6-pillar framework to their OWN brand's current state first (a self-assessment is the canonical day-1 use case), then expand to specific data sources + supplier audits + certifications later; this avoids the "I don't have data sources yet so I'll defer" trap. (b) **greenwashing risk is structural** — vague claims ("eco-friendly", "sustainable", "natural") without measurable outcomes are now FTC-enforceable per the 2023 Green Guides update + EU Empowering Consumers directive; the framework forces measurable outcomes (tons CO2e shipped / % recycled / % living-wage / % recycled-input / $ donated / B-Corp-score). (c) **voice-driven override columns matter MORE than the headline numbers** — Sustainable brands reporting "100 tons CO2e offset" + Default brands reporting "100 tons CO2e offset" are NOT the same report; the Sustainable variant leads with the mission + the certification + the audit-trail, while the Default variant translates to "your last shipment = 5 miles of driving offset." (d) **the report is a strategy tool, not a vanity metric** — picking the most flattering metrics (carbon offset, recycled materials %) and ignoring the harder ones (labor conditions in supplier factories, packaging recyclability at end-of-life) is the greenwashing trap; all 6 pillars must be reported, even if the number is unflattering. (e) **regulatory pressure is increasing** — the EU CSRD applies to brands with €150M+ revenue (phased through 2029); the US SEC climate-disclosure rule applies to public companies; California's SB-253 / SB-261 apply to brands with $1B+ / $500M+ revenue doing business in CA; the framework is the on-ramp for compliance when the operator crosses those thresholds. (f) **data sources are 80% of the work** — picking the right data source (Shopify Planet for carbon / Bluesign for materials / Fair Trade USA for labor) costs $0–$500/mo; the framework is the durable artifact; data sources plug in later. **The framework ships before data sources, just like Asset 07's framework shipped before competitors were named.**

## The 6-pillar impact-reporting framework

The canonical impact-reporting framework scores any DTC brand across 6 pillars on a 1–5 maturity scale. Each pillar has 5 voice-driven override columns (Default / Luxury / Sustainable / Gen-Z / B2B) so that operators using Asset 02's voice framework report each pillar differently.

**Maturity scale (universal across all 6 pillars):**

- **1 = Awareness** — operator has thought about the pillar but has no data, no commitment, no public claim. This is the baseline.
- **2 = Commitment** — operator has made a public commitment (e.g. "carbon-neutral by 2030") but has no data, no measurement, no certification. This is the greenwashing-trap zone.
- **3 = Measured** — operator has baseline data + is tracking year-over-year improvement. The pillar is verifiable, not aspirational.
- **4 = Certified** — operator has 3rd-party certification (Climate Neutral Certified / B Corp / Bluesign / GOTS / Fair Trade USA / How2Recycle) + public audit report. The pillar is no longer self-reported.
- **5 = Industry-leading** — operator is best-in-class on this pillar; the certification is current + the audit report is public + the year-over-year improvement is published. The pillar is a competitive moat.

**Default case for $0–$50k/mo GMV:** Self-assessment only (score your own brand on all 6 pillars). The framework is the durable artifact; data sources + certifications can be added later when GMV justifies the audit cost (most certifications are $1k–$10k audit + $500–$5k/yr maintenance).

**Default case for $50k–$500k/mo GMV:** Self-assessment + 2nd-party data sources (Shopify Planet for carbon + supplier ESG reports for materials + How2Recycle for packaging) + at least 1 third-party certification (Climate Neutral Certified is the cheapest at ~$5k/yr; B Corp is the most expensive at ~$25k audit + $1k/yr).

**Default case for $500k+/mo GMV:** Self-assessment + 3rd-party data sources (EcoCart + Patch for carbon / Bluesign + GOTS for materials / Fair Trade USA for labor) + 2–3 third-party certifications (Climate Neutral + B Corp + Fair Trade USA) + CSRD-ready reporting structure (XBRL-tagged metrics for EU compliance).

### Pillar 1 — Carbon (shipping + operations + supplier)

**What this pillar measures:** Total greenhouse-gas emissions attributable to the brand's operations, normalized per shipment (tons CO2e per shipment) and per dollar revenue (tons CO2e per $k revenue). The strongest carbon pillar reports both numerator (total emissions) and denominator (per-unit + per-revenue); the weakest carbon pillar reports only "we offset X tons" without measurement.

**Sub-questions (paste-ready scoring rubric):**
1. Has the operator measured their Scope 1 + Scope 2 + Scope 3 emissions (the GHG Protocol's three scopes — direct / energy / value-chain) within the last 12 months?
2. Is the per-shipment carbon footprint published on PDPs or cart page (per Asset 07 Dimension 5 Shipping + Move #3 checkout audit)?
3. Is carbon offset purchased for the residual footprint (after measurement + reduction)? If yes, what is the offset provider (Patch / Gold Standard / Verra / Climate Neutral Certified)?
4. Is the operator on a public Net-Zero timeline with year-over-year progress reports (per CSRD + SEC climate-disclosure readiness)?
5. Does the operator avoid the "carbon-neutral by 2030" greenwashing trap (commitment without measurement) per the FTC Green Guides + EU Empowering Consumers directive?

**What moves this pillar:** Shopify Planet carbon-app (free for Shopify merchants; auto-calculates per-shipment carbon + offers offset at checkout) + EcoCart (Shopify app; $0–$500/mo based on offset volume; offers per-shipment opt-in for customers) + Patch ($0.10–$0.50 per offset; API-first; per shipment / per checkout / per product) + Climate Neutral Certified certification (~$5k/yr; one-time audit + annual reporting).

**Default vs override scoring (1–5 per voice profile):**

| Voice profile | Score 5 = Industry-leading signature | Score 1 = Awareness signature |
|---|---|---|
| **Default** | Customer-friendly per-shipment carbon published on PDP ("this shipment = 2.3 lbs CO2e, offset via Shopify Planet") + per-customer annual impact update via Klaviyo | Generic "we care about the environment" without measurement |
| **Luxury** | Heritage + scarcity + provenance-led carbon reporting ("the Florence atelier has been solar-powered since 2018; the leather tannery offsets 100% of Scope 3 via partner X") | Mass-market offset language ("we plant a tree for every order") without audit |
| **Sustainable** | Mission + numbers + certification-led carbon reporting ("Climate Neutral Certified since 2021; 2024 Scope 1+2+3 = X tons CO2e; -Y% YoY; full audit at /impact-report-2024.pdf") | Vague mission language ("we're committed to sustainability") without numbers |
| **Gen-Z** | Humor + accountability + transparency-led carbon reporting ("we shipped Y tons CO2e last year = Z flights; we offset W; here's our audit and here's where we're still shit") | Performative activism without data ("we're saving the planet one package at a time") |
| **B2B** | Compliance + ROI + audit-trail-led carbon reporting ("CSRD-aligned Scope 1+2+3 disclosure; XBRL-tagged metrics; audit firm X; CSRD report Q2 2025") | Vague "sustainability commitment" without audit firm or CSRD readiness |

### Pillar 2 — Materials (sourcing + recycled content + end-of-life)

**What this pillar measures:** The % of product materials that are recycled / organic / renewable / certified, plus the end-of-life pathway (recyclable / compostable / take-back program). The strongest materials pillar reports per-product material breakdown; the weakest reports only "eco-friendly materials" without specification.

**Sub-questions (paste-ready scoring rubric):**
1. Does the operator have a per-product material breakdown published on PDPs (e.g. "60% organic cotton, 40% recycled polyester, GOTS-certified")?
2. Does the operator have a chain-of-custody certification for at least one material category (GOTS for organic cotton / GRS for recycled / FSC for paper / leather working group for leather)?
3. Does the operator measure end-of-life (How2Recycle label on packaging / take-back program for used products / composting instructions for compostable items)?
4. Does the operator avoid the "natural / eco-friendly / green" greenwashing-trap language without specification (per FTC Green Guides + EU Empowering Consumers)?
5. Does the operator track year-over-year improvement in recycled / organic / certified % (a measured-improvement score, not just a static number)?

**What moves this pillar:** Bluesign certification (~$3k–$10k/yr; chemical-management certification for textile mills) + GOTS certification (~$2k–$5k/yr; organic textile chain-of-custody) + GRS certification (~$2k–$5k/yr; recycled-content chain-of-custody) + Oeko-Tex Standard 100 (~$1k–$3k/yr; harmful-substances certification) + How2Recycle label (free; standardized recyclability labeling for packaging).

**Default vs override scoring (1–5 per voice profile):**

| Voice profile | Score 5 = Industry-leading signature | Score 1 = Awareness signature |
|---|---|---|
| **Default** | Per-product material breakdown on PDP ("60% organic cotton, GOTS-certified") + How2Recycle label on packaging | Generic "natural materials" without specification |
| **Luxury** | Heritage + provenance-led material reporting ("the leather is sourced from the same Florence tannery since 1948; full chain-of-custody at /provenance") | Mass-market "eco" claims without provenance |
| **Sustainable** | Mission + numbers + certification-led material reporting ("100% GOTS organic cotton since 2020; GRS recycled packaging; full material audit at /impact-report-2024.pdf") | Vague "eco-friendly materials" without certification |
| **Gen-Z** | Transparency + accountability-led material reporting ("we use 60% organic + 40% conventional; here's why, here's our transition plan, here's the audit") | Performative "we're 100% sustainable" without nuance |
| **B2B** | Compliance + chain-of-custody-led material reporting ("GOTS + GRS + Bluesign certifications; full chain-of-custody documentation available for procurement teams") | Vague "sustainable materials" without audit trail |

### Pillar 3 — Labor (living wage + supplier conditions + DEI)

**What this pillar measures:** Whether workers across the operator's supply chain earn a living wage (vs minimum wage) AND work in audited safe conditions. The strongest labor pillar publishes per-factory wage data + living-wage gap analysis; the weakest publishes only "we care about our workers" without factory disclosure.

**Sub-questions (paste-ready scoring rubric):**
1. Has the operator published a supplier factory list (full or tier-1 only — both are valid)?
2. Does the operator measure living-wage gap (% of supplier workers earning below the local living-wage benchmark, per the WageIndicator Foundation or MIT Living Wage)?
3. Does the operator have a 3rd-party labor audit (Fair Trade USA / B Corp / Sedex / SA8000) at least annually?
4. Does the operator avoid the "we audit our suppliers" greenwashing-trap (audits can be scheduled + announced + bribery-tolerant; the framework asks for unannounced + 3rd-party + worker-interview-based audits)?
5. Does the operator track year-over-year improvement in living-wage % + factory-condition score?

**What moves this pillar:** Fair Trade USA certification ($3k–$10k/yr + per-product certification fee) + B Corp certification ($25k audit + $1k/yr; covers labor + environment + governance) + Sedex SMETA audit ($1k–$5k/yr per factory) + WageIndicator Foundation data (free; per-country living-wage benchmark) + supplier wage statements (self-reported; required for Fair Trade + B Corp).

**Default vs override scoring (1–5 per voice profile):**

| Voice profile | Score 5 = Industry-leading signature | Score 1 = Awareness signature |
|---|---|---|
| **Default** | Customer-friendly labor disclosure ("every factory in our supply chain is Fair Trade USA-certified; living wage verified annually") | Generic "ethical manufacturing" without factory list |
| **Luxury** | Heritage + artisan-led labor reporting ("every piece is hand-stitched by artisans earning X% above living wage at our partner atelier in Florence") | "Artisan-made" without wage disclosure |
| **Sustainable** | Mission + audit-led labor reporting ("100% Fair Trade USA supply chain; 2024 living-wage gap = X%; full factory list at /factories") | Vague "ethical" claims without factory disclosure |
| **Gen-Z** | Transparency + accountability-led labor reporting ("here's every factory in our supply chain; here's the wage data; here's what we're fixing") | Performative activism without factory disclosure |
| **B2B** | Compliance + audit-trail-led labor reporting ("SA8000 + Sedex SMETA audited; full audit reports available under NDA for procurement teams") | Vague "ethical supplier" claims without audit firm |

### Pillar 4 — Packaging (recycled input + recyclability + end-of-life)

**What this pillar measures:** Whether the operator's packaging uses recycled input (post-consumer recycled content) AND is recyclable / compostable / reusable at end-of-life. The strongest packaging pillar reports per-packaging-component recyclability + FSC certification; the weakest reports only " recyclable packaging" without per-component specification.

**Sub-questions (paste-ready scoring rubric):**
1. Does the operator measure per-packaging-component recycled input (% PCR in mailers / boxes / tissue / stickers / inserts)?
2. Does the operator use How2Recycle labels on every packaging component (standardized recyclability labeling for US consumers)?
3. Does the operator have FSC certification for paper-based packaging (FSC mix / FSC recycled)?
4. Does the operator avoid the "100% recyclable" greenwashing-trap (per the FTC Green Guides — most curbside programs don't accept multi-material mailers or poly-coated paper)?
5. Does the operator track year-over-year improvement in PCR % + recyclability % + reusable-packaging %?

**What moves this pillar:** How2Recycle label (free; standardized US recyclability labeling) + FSC certification ($1k–$5k/yr; chain-of-custody for paper packaging) + noissue ($0.10–$0.30/unit; FSC-certified + compostable + soy-ink mailers + tissue + stickers) + EcoEnclose ($0.10–$0.30/unit; 100% PCR mailers + boxes) + returnable-packaging program (deposit-based reusable mailers, e.g. Returnity).

**Default vs override scoring (1–5 per voice profile):**

| Voice profile | Score 5 = Industry-leading signature | Score 1 = Awareness signature |
|---|---|---|
| **Default** | Per-component PCR published on PDP ("100% PCR mailer + FSC tissue + How2Recycle label on every component") | Generic " recyclable packaging" without per-component |
| **Luxury** | Heritage + unboxing-led packaging reporting ("the unboxing uses FSC tissue + soy-ink ribbon + reusable wooden box; designed for keepsake storage") | "Premium packaging" without recyclability specification |
| **Sustainable** | Mission + numbers + certification-led packaging reporting ("100% PCR + FSC + How2Recycle since 2020; 2024 PCR % = X; full audit at /impact-report-2024.pdf") | Vague "eco packaging" without certification |
| **Gen-Z** | Humor + transparency-led packaging reporting ("our mailer is 100% PCR + the tissue is FSC + the sticker is compostable; the 'thank you' card is the only thing not yet — working on it") | Performative "eco" claims without data |
| **B2B** | Compliance + audit-trail-led packaging reporting ("FSC + How2Recycle + EPR-compliant in EU markets; full documentation for procurement teams") | Vague "sustainable packaging" without certification |

### Pillar 5 — Community (1% for the Planet + donations + impact-creator partnerships)

**What this pillar measures:** Whether the operator gives back to the community through donations (1% for the Planet / B Corp required donations) + impact-creator partnerships (Asset 03's UGC creators aligned with the mission) + customer-funded donations (Move #8 loyalty points redeemable for donations). The strongest community pillar publishes per-cause donation totals + recipient organizations; the weakest publishes only "we give back" without amounts.

**Sub-questions (paste-ready scoring rubric):**
1. Does the operator publish annual donation totals per cause (e.g. "$X donated to Y organization in 2024")?
2. Does the operator have a 1% for the Planet certification (1% of revenue donated to environmental causes; $500/yr membership + 1% revenue commitment)?
3. Does the operator partner with impact-aligned creators (Asset 03 UGC creators whose personal mission aligns with the brand's mission)?
4. Does the operator allow customer-funded donations (Move #8 loyalty points redeemable for $X donation OR cart-total donation toggle at checkout)?
5. Does the operator track year-over-year improvement in donation $ + donation recipient count + customer-funded donation $?

**What moves this pillar:** 1% for the Planet certification ($500/yr + 1% revenue commitment) + B Corp certification (requires community-impact component) + Asset 03 UGC creator partnerships (impact-creator brief template) + Move #8 loyalty impact-rewards (points-for-donations rule) + Move #3 checkout audit (donation toggle at checkout).

**Default vs override scoring (1–5 per voice profile):**

| Voice profile | Score 5 = Industry-leading signature | Score 1 = Awareness signature |
|---|---|---|

| **Luxury** | Heritage + cause-led community reporting ("since 2018, 1% of every Florence piece has funded artisan apprenticeship in the region; 2024 = $X to Y program") | "Charitable giving" without recipient disclosure |
| **Sustainable** | Mission + numbers + certification-led community reporting ("1% for the Planet member since 2020; 2024 donations = $X to Y, Z, W; full ledger at /impact-report-2024.pdf") | Vague "community impact" without amounts |
| **Gen-Z** | Transparency + accountability-led community reporting ("here's every org we donated to in 2024; here's the amount; here's what they did with it; here's where we're still figuring it out") | Performative activism without recipient disclosure |

### Pillar 6 — Certification (B Corp + Climate Neutral Certified + chain-of-custody)

**What this pillar measures:** Whether the operator has third-party certifications that validate the other 5 pillars. The strongest certification pillar has 3+ current certifications (B Corp + Climate Neutral + chain-of-custody); the weakest has 0 certifications (self-reported only).

**Sub-questions (paste-ready scoring rubric):**
1. Does the operator have B Corp certification (the gold-standard sustainability certification; covers environment + labor + governance + community + customer)?
2. Does the operator have Climate Neutral Certified (the gold-standard carbon certification; annual audit + brand-level certification)?
3. Does the operator have at least one chain-of-custody certification (GOTS / GRS / FSC / Fair Trade USA / Bluesign)?
4. Are all certifications current (annual renewal + audit completed within the last 12 months)?
5. Does the operator avoid the "B Corp pending" greenwashing-trap (claiming B Corp benefits without certification; the framework treats "B Corp pending" as Score 2 = Commitment, not Score 4 = Certified)?

**What moves this pillar:** B Corp certification ($25k one-time audit + $1k/yr maintenance) + Climate Neutral Certified (~$5k/yr + annual audit) + GOTS / GRS / FSC / Fair Trade USA / Bluesign (chain-of-custody; $1k–$10k/yr each) + annual certification renewal cycle (each certification requires annual audit + renewal).

**Default vs override scoring (1–5 per voice profile):**

| Voice profile | Score 5 = Industry-leading signature | Score 1 = Awareness signature |
|---|---|---|
| **Default** | "B Corp Certified + Climate Neutral Certified" prominently on PDP + landing page + footer | "B Corp pending" or "aspiring B Corp" without certification |
| **Luxury** | Heritage + certification-led reporting ("B Corp + Climate Neutral + Fair Trade USA certified; full certifications at /certifications") | "Certified sustainable" without specifying which certifications |
| **Sustainable** | Mission + numbers + certification-led reporting ("B Corp (score 95.4) + Climate Neutral + GOTS + Fair Trade USA; all current; full audit reports at /impact-report-2024.pdf") | Vague "certified" claims without listing certifications |
| **Gen-Z** | Transparency + accountability-led reporting ("we have 3 certifications; here's what each one means; here's what they DON'T cover; here's what we're working toward") | Performative certification claims without listing |
| **B2B** | Compliance + audit-trail-led reporting ("B Corp (score 95.4) + Climate Neutral + GOTS + Fair Trade USA; all audit reports available under NDA for procurement teams") | Vague "certified" claims without audit availability |

**Default stack for $50k–$500k/mo GMV:** Above + Climate Neutral Certified ($5k/yr) + Fair Trade USA ($3k–$10k/yr) + Bluesign or GOTS ($3k–$10k/yr) = $11k–$25k/yr = the standard DTC impact-reporting stack.

## 10 common impact-reporting pitfalls with corrective `Fix:` lines

**Pitfall #1 — Greenwashing-trap language ("eco-friendly", "sustainable", "natural") without measurable outcomes.** Symptom: PDP or landing page says "eco-friendly materials" or "sustainable practices" without specifying what the materials are, what the practices are, or what the measurable outcomes are. Fix: replace every greenwashing-trap phrase with a measurable-outcome phrase per the FTC Green Guides + EU Empowering Consumers directive. "Eco-friendly materials" → "60% organic cotton, GOTS-certified." "Sustainable practices" → "100% carbon-offset via Shopify Planet since 2022."

**Pitfall #2 — Commitment without measurement ("carbon-neutral by 2030" without per-year data).** Symptom: brand publishes a Net-Zero-by-2030 timeline without per-year emissions data + reduction plan. This is the canonical greenwashing trap — commitment is aspirational; measurement is verifiable. Fix: ship a per-year emissions report (Scope 1 + Scope 2 + Scope 3) with year-over-year reduction %, then publish the Net-Zero-by-2030 plan against the per-year data. Without per-year data, the Net-Zero timeline is marketing, not commitment.

**Pitfall #3 — Carbon offset without reduction (buying offsets instead of reducing emissions).** Symptom: brand purchases carbon offsets equal to total emissions without first reducing emissions. Offsets are the residual after reduction; reduction must come first. Fix: publish per-year emissions (Scope 1+2+3) + per-year reduction % + per-year offset purchased (in that order). The framework's reporting order is reduce → measure → offset.

**Pitfall #4 — Carbon-neutral certification without per-shipment measurement.** Symptom: brand has Climate Neutral Certified (or similar) but doesn't publish per-shipment carbon on PDPs. Certification is at the brand level; per-shipment measurement is at the product level — both are required. Fix: use Shopify Planet or EcoCart to publish per-shipment carbon on every PDP ("this shipment = 2.3 lbs CO2e, offset via Shopify Planet").

**Pitfall #5 — Chain-of-custody certification expired or never renewed.** Symptom: PDP or landing page says "GOTS-certified" but the certification has expired (most certifications are annual). Expired certifications are the silent greenwashing trap — operators forget to renew; the PDP still says "certified" for 6+ months. Fix: annual renewal calendar + automated email reminder 60 days before each certification expires. The framework's Pillar 6 (Certification) sub-question #4 requires "all certifications current."

**Pitfall #6 — Fair Trade / B Corp / "ethical" claim without factory list.** Symptom: brand says "ethical manufacturing" or "Fair Trade certified" but doesn't publish a factory list. Without factory list, the claim is unverifiable. Fix: publish full supplier factory list (at minimum tier-1, ideally tier-2 + tier-3 for full chain-of-custody). B Corp + Fair Trade USA both require factory disclosure as part of certification.

**Pitfall #7 — Recyclable packaging claim without per-component specification.** Symptom: brand says " recyclable packaging" but the mailer is multi-material (paper + plastic lining) which is NOT curbside-recyclable in most US programs. Per the FTC Green Guides, " recyclable" claims require that the packaging is actually recyclable in the majority of consumer programs. Fix: use How2Recycle labels on every packaging component (per Pillar 4 sub-question #2). The label specifies per-component recyclability ("Please recycle this paper mailer; the plastic film is not currently recyclable").

**Pitfall #8 — "1% for the Planet" claim without annual donation report.** Symptom: brand says "we donate 1% to environmental causes" without publishing the annual donation amount + recipient organizations. Without annual report, the claim is unverifiable. Fix: publish annual donation report per Pillar 5 sub-question #1 ("$X donated to Y organization in 2024; 1% for the Planet member since 2020").

**Pitfall #9 — Impact report hidden on a non-prominent URL ("/impact-report-2024.pdf" not linked from footer or PDP).** Symptom: brand publishes an impact report but doesn't link it from PDP / landing page / footer. The report exists but is undiscoverable. Fix: link impact report from footer ("Impact") + PDP ("Carbon footprint: 2.3 lbs CO2e; full impact report at /impact-report-2024.pdf") + landing page ("B Corp certified + Climate Neutral + view our 2024 impact report").

**Pitfall #10 — Impact report not updated annually (the report is from 2022, brand is on 2025).** Symptom: brand publishes impact report once and never updates. The report goes stale; customers / partners / regulators see the old data. Fix: annual update cadence per Asset 04's Q1 (Earth Day / B Corp month) macro shape. Set a calendar reminder: impact report updated by April 22 (Earth Day) every year.

## 5 verification gates

**Gate A — Carbon-verification via Shopify Planet API + supplier energy invoices.** Recipe: (1) Shopify Planet app installed + per-shipment carbon calculated + offset at checkout enabled (free for Shopify merchants); (2) supplier energy invoices collected for top-10 suppliers (manufacturing facilities + warehouses + offices) + Scope 1 + Scope 2 calculated per GHG Protocol; (3) Scope 3 calculated via Shopify Planet (covers shipping) + supplier-specific estimates for inbound logistics + employee commuting + product end-of-life; (4) annual audit by Climate Neutral Certified or equivalent 3rd-party body. Pass criteria: per-shipment carbon published on every PDP + annual Scope 1+2+3 report published + per-year reduction tracked.

**Gate B — Materials-verification via supplier certificates + GOTS / GRS / Oeko-Tex chain-of-custody.** Recipe: (1) supplier material certificates collected for every material category (organic cotton / recycled polyester / FSC paper / etc.); (2) per-product material breakdown published on PDP ("60% organic cotton, GOTS-certified"); (3) GOTS / GRS / Oeko-Tex certification maintained annually for at least one material category; (4) annual audit by certification body. Pass criteria: per-product material breakdown published on every PDP + at least one chain-of-custody certification current.

**Gate C — Labor-verification via Fair Trade USA audit + supplier wage statements.** Recipe: (1) supplier factory list published (tier-1 minimum, tier-2 + tier-3 for B Corp + Fair Trade USA); (2) supplier wage statements collected for every factory (base wage + benefits + overtime); (3) living-wage gap calculated per WageIndicator Foundation benchmark (% of workers earning below local living wage); (4) Fair Trade USA or B Corp or Sedex SMETA audit annually. Pass criteria: factory list published + living-wage gap published annually + 3rd-party labor audit current.

**Gate D — Packaging-verification via How2Recycle label audit + FSC certificates.** Recipe: (1) How2Recycle labels applied to every packaging component (mailer / box / tissue / sticker / insert); (2) FSC certification maintained annually for paper-based packaging (FSC mix / FSC recycled); (3) PCR % measured for every mailer + box (% post-consumer recycled content); (4) annual packaging audit by 3rd-party body. Pass criteria: How2Recycle labels on every component + FSC certification current + PCR % published per packaging component.

**Gate E — Community-verification via 1% for the Planet donation receipts + Move #8 loyalty-impact-rewards ledger.** Recipe: (1) 1% for the Planet certification maintained annually ($500/yr + 1% revenue commitment); (2) annual donation report published with per-cause donation totals + recipient organizations; (3) Move #8 loyalty-impact-rewards rule wired (customers can redeem points for carbon-offset shipping / B-Corp donation / 1%-matching donation); (4) Move #3 checkout donation toggle wired (customer can add $1/$3/$5 to cart-total donation). Pass criteria: annual donation report published + 1% for the Planet certification current + loyalty-impact-rewards rule live + checkout donation toggle live.

**Gate F (bonus) — Certification-verification.** Recipe: (1) B Corp + Climate Neutral + chain-of-custody certifications maintained annually (each has its own audit + renewal cycle); (2) certification logos + certification numbers + certification expiration dates linked from footer + landing page + PDP; (3) annual certification audit reports published (or available under NDA for B2B). Pass criteria: B Corp + Climate Neutral + at least one chain-of-custody certification all current + audit reports published (or available for B2B procurement teams).

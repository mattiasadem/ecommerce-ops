## The 5 voice profiles (canonical from `assets/02-brand-voice.md`)

The canonical 5-voice framework ships in `assets/02-brand-voice.md` §The 5-dimension voice framework. Reuse it verbatim for every wholesale-listing copy below:

- **Default** — confident + plain + lightly irreverent. Use for **mass-market DTC** voice brands.
- **Luxury** — refined + quiet + maximalist grammar. Use for **luxury DTC** voice brands.
- **Sustainable** — warm + earnest + educational. Use for **clean-beauty / sustainable** voice brands.
- **Gen-Z** — high-energy + lowercase + meme-aware. Use for **Gen-Z** voice brands.
- **B2B** — precise + benefit-led + no jokes. Use for **professional / hospitality / wellness-practitioner** retail buyers (the default wholesale-buyer voice regardless of brand-DTC-voice).

**Wholesale-listings rule:** the **wholesale-listing voice defaults to B2B** (precise + benefit-led) regardless of DTC-brand-voice. The 5-voice columns below show how to ADAPT the wholesale-listing copy when the brand's DTC voice carries through (e.g. a Luxury-DTC brand writing wholesale-listings still sounds Luxury, not B2B-default — the per-marketplace retailer-buyer reads "branded" listings even on Faire). For most Path-B brands, the listings read B2B-default with a brand-DTC-voice-overlaid headline.

## The 6 SKU archetypes (canonical from `research/10-b2b-wholesale.md` Pillar 1)

Each archetype maps to a wholesale-channel-fit tier per Faire 2024 + Ankorstore 2024 benchmarks. The 6 archetypes cover 90%+ of all B2B-eligible DTC SKUs:

- **Consumables** — food / beverage / supplements / vitamins / pet-treats. Highest B2B-fit; 60-80% reorder-rate by Year-2 per Salesforce B2B Commerce 2024.
- **Apparel** — clothing / accessories / footwear. Mid B2B-fit; 40-60% reorder-rate; casepack-size matters most.
- **Home** — decor / kitchen / linens / candles. Mid B2B-fit; 35-50% reorder-rate; lifestyle-buyer dominant.
- **Beauty** — skincare / haircare / cosmetics / fragrance. High B2B-fit for clean-beauty retail; 50-70% reorder-rate.
- **Specialty-food** — artisanal / organic / gourmet / single-origin. Highest B2B-fit for Ankorstore + Faire + RSP/KeHE/UNFI; 70-85% reorder-rate.
- **Pet-supplies** — food / treats / toys / wellness. Mid B2B-fit; 45-60% reorder-rate; pet-boutique-buyer dominant.

## The 4 canonical wholesale marketplaces

Each marketplace has distinct voice + format conventions per Faire 2024 + Tundra 2024 + Ankorstore 2024 + Handshake 2024 benchmarks:

- **Faire** — 25%-first-order-commission + 15%-reorder-commission. **Discovery-led listings**: hero-image + lifestyle-image + 3-5 bullet benefits + retail-MSRP + wholesale-price + lead-time. Default voice = **B2B-default** with brand-DTC-voice-overlaid headline.
- **Tundra** — 0% commission for $1M+ GMV brands. **Bulk-buyer-led listings**: 6 bullet benefits + wholesale-tier pricing (50/55/60% off MSRP at 1-carton / 5-carton / 20-carton) + lead-time + casepack-spec. Default voice = **B2B-precise**.
- **Ankorstore** — 8-15% commission. **EU-UK-AU-discovery-led listings**: hero-image + 5-bullet EU-buyer-targeted benefits + EU-wholesale-price (EUR) + lead-time + EU-broker-fee-disclosure. Default voice = **B2B-precise** with EU-buyer-language conventions (e.g. "trade customer" / "RRP" / "MOQ").
- **Handshake (Shopify B2B)** — 0% commission + Shopify-native. **Catalog-led listings**: product-name + wholesale-price + casepack-size + lead-time + reorder-button. Default voice = **B2B-default**; brand-DTC-voice is preserved on the listing because Shopify renders the DTC-page wholesale-priced.

## 12 common B2B-wholesale-kit pitfalls (with corrective Fix lines)

### A. Wholesale-pricing-calculator mismatch (3 pitfalls)

**Pitfall #1 — Wholesale-price-set at 60% off MSRP without margin-defense.** Brand sets wholesale at 60% off MSRP without checking the canonical 6-tier wholesale-discount matrix per `research/10-b2b-wholesale.md` Pillar 2 §6-tier wholesale-discount matrix. The result: gross-margin erosion of 5-15pp per Faire 2024 + Ankorstore 2024 benchmarks. **Fix:** run the canonical `scripts/b2b_wholesale_unit_economics.py` (planned, does not yet exist) to validate wholesale-price ≤ canonical-tier-max for the brand's voice-profile (35% off luxury / 40% off sustainable / 50% off default / 55% off high-volume / 60% off distributor).

**Pitfall #2 — NET-30-terms-default without pre-pay-with-7-day-discount.** Brand offers NET-30 without the canonical pre-pay-with-7-day-discount per Stripe B2B 2024 benchmarks. The result: 4-7% effective-financing-cost on wholesale-AR; brands with $1M+ B2B-AR should consider Stripe B2B Capital for 1-3% effective rate. **Fix:** add the pre-pay-with-7-day-discount line to the NET-30 terms-card per the canonical template above.

**Pitfall #3 — Casepack-size mismatch with Faire / Tundra / RSP defaults.** Brand sets casepack-size at 6 (default Shopify) without checking Faire's 12-unit-default / Tundra's 24-unit-default / RSP's 48-unit-default per Faire 2024 + Tundra 2024 + RSP 2024 benchmarks. The result: casepack-mismatch cost-stack of 5-15% retail-broker-fees. **Fix:** align casepack-size to the channel's default per the per-channel MOQ + casepack matrix above.

### B. MAP-policy-page missing or weak (3 pitfalls)

**Pitfall #4 — No MAP-policy-page published.** Brand launches wholesale without a MAP-policy-page per Faire 2024 + Ankorstore 2024 benchmarks. The result: 30-50% DTC-traffic erosion as authorized retailers undercut DTC pricing on Amazon / Google Shopping. **Fix:** publish the canonical MAP-policy-page template above; require all wholesale accounts to acknowledge before first order.

**Pitfall #5 — MAP-policy-page without 3-strike enforcement.** Brand has MAP-policy-page but doesn't enforce 3-strike per Faire 2024 benchmarks. The result: MAP-violations persist; brands without internal-enforcement-team lose 50-60% of MAP-policy-effectiveness. **Fix:** add the 3-strike-enforcement clause to the MAP-policy-page; designate a wholesale-relations-team member as the MAP-enforcement owner.

**Pitfall #6 — MAP-policy-page without Sherman-Antitrust-Act-RPM-policy-per-se-exemption citation.** Brand has MAP-policy-page but doesn't cite the Sherman-Antitrust-Act-RPM-policy-per-se-exemption. The result: legal vulnerability if a retailer challenges MAP-policy in court. **Fix:** add the Sherman-Antitrust-Act-RPM-policy-per-se-exemption citation to the MAP-policy-page per the canonical template above.

### C. Distributor-onboarding-pack incomplete (3 pitfalls)

**Pitfall #7 — Missing GS1-Company-Prefix certificate.** Brand sends the 8-prereq RFQ brief without GS1-Company-Prefix certificate per GS1 2024 benchmarks. The result: distributor-onboarding stalls 6-12 weeks waiting for GS1-cert ($250/yr + 6-week lead time). **Fix:** include the GS1-Company-Prefix certificate in the RFQ attachments; if not yet issued, note "GS1-Company-Prefix pending — 6-week lead time" + provide temporary SKU-barcode-via-Shopify.

**Pitfall #8 — Missing product-insurance certificate.** Brand sends the 8-prereq RFQ brief without product-insurance certificate. The result: distributor-onboarding stalls waiting for product-liability-insurance renewal ($1M+ per occurrence + $2M+ aggregate). **Fix:** include the product-insurance certificate in the RFQ attachments; if not yet issued, note "product-insurance pending — 2-week lead time".

**Pitfall #9 — Missing resale-certificate per Streamlined-Sales-Tax-Project.** Brand sends the 8-prereq RFQ brief without resale-certificate. The result: distributor-onboarding stalls waiting for state-by-state resale-cert registration. **Fix:** include the resale-certificate in the RFQ attachments; if not yet issued, note "resale-certificate pending — 2-4 week lead time per state".

### D. Corporate-gifting-catalog missing (2 pitfalls)

**Pitfall #10 — No corporate-gifting-catalog.** Brand launches wholesale without a corporate-gifting-catalog per Faire 2024 + Ankorstore 2024 benchmarks. The result: missing 15-25% of Year-2 B2B revenue (corporate-gifting is a major B2B-buyer use-case). **Fix:** publish the canonical corporate-gifting-catalog template above; promote via Faire + Tundra + Handshake + Ankorstore B2B-buyer-cadence.

**Pitfall #11 — Corporate-gifting-catalog without custom-ribbon + handwritten-card upsell.** Brand publishes corporate-gifting-catalog without the canonical custom-ribbon + handwritten-card upsell per Faire 2024 benchmarks. The result: 30-40% lower average-order-value vs brands that offer custom-ribbon + handwritten-card. **Fix:** add custom-ribbon + handwritten-card upsell lines to the corporate-gifting-catalog per the canonical template above.

### E. Klaviyo / Triple Whale wiring missing (1 pitfall)

**Pitfall #12 — B2B-flow without Triple Whale UTM + cohort-overlay.** Brand launches B2B-flow without the canonical Triple Whale UTM + cohort-overlay per `research/10-b2b-wholesale.md` Pillar 1. The result: B2B-buyer-cohort-LTV is unmeasurable; brand under-prices B2B cannibalization by 30-50%. **Fix:** add the canonical Triple Whale UTM to every CTA + activate the B2B-buyer cohort-overlay on Triple Whale's cohort-LTV dashboard.

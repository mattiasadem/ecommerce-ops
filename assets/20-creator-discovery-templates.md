## The Triple-Whale-creator-cohort-overlay-Wire-spec (Pillar 5 attribution-merge substrate)

The canonical Triple-Whale-creator-cohort-overlay-Wire-spec defines the data-warehouse-schema for creator-cohort-segmentation + creator-cohort-attribution-merge + 5-way-comparison-cycle-data per the canonical `research/12-creator-economy-expansion.md` Pillar 5.

### Creator-cohort-segmentation-rules

- **Micro-cohort:** creators with 1k-10k followers (60% of creator-pool per creator-tier-mix-baseline).
- **Mid-tier-cohort:** creators with 10k-100k followers (30% of creator-pool).
- **Macro-cohort:** creators with 100k+ followers (10% of creator-pool).
- **Per-voice-cohort:** Default / Luxury / Sustainable / Gen-Z / B2B (5 voice-cohorts per creator-tier).
- **Per-channel-cohort:** TikTok / Instagram / YouTube / Pinterest / Podcast / Substack / Newsletter / UGC (8 channel-cohorts).
- **Per-product-cohort:** per-SKU-archetype (Beauty / Fashion / Home / Food-and-Beverage + Pet + Health / Lifestyle / B2B).
- **Per-cohort-LTV-window:** 30-day / 60-day / 90-day / 180-day / 365-day windows (5 LTV-windows per cohort).

### Creator-cohort-attribution-merge-rules

- Triple-Whale-pixel captures creator-handle + creator-cohort + creator-channel + creator-content-piece on every site-visit.
- Klaviyo-event captures creator-handle on every email-open + email-click + SMS-click.
- Recharge (or Skio / Stay AI) captures creator-handle on every subscription-event.
- Smile.io captures creator-handle on every loyalty-event.
- Per-creator-cohort-LTV is computed as: sum-of-revenue-per-cohort-customer / count-of-cohort-customers.

### 5-way-comparison-cycle-data-warehouse-schema

```sql
-- 5-way-comparison-table: creator-economy-cohort-LTV vs organic-DTC-cohort-LTV vs paid-Meta-cohort-LTV vs affiliate-program-cohort-LTV [Move #15] vs TikTok-Shop-cohort-LTV [Move #15.x]
SELECT
  cohort_type,                    -- 'creator_economy' / 'organic_dtc' / 'paid_meta' / 'affiliate_program' / 'tiktok_shop'
  ltv_window,                     -- '30d' / '60d' / '90d' / '180d' / '365d'
  voice_profile,                  -- 'default' / 'luxury' / 'sustainable' / 'gen_z' / 'b2b'
  cohort_size,                    -- count of customers in this cohort
  total_revenue,                  -- total revenue from this cohort
  avg_ltv,                        -- average LTV per customer in this cohort
  -- 5-way-comparison-ratio: cohort-LTV / benchmark-cohort-LTV
  ltv_ratio_vs_organic_dtc,       -- creator_economy / organic_dtc (the canonical 2-4× Path B baseline)
  ltv_ratio_vs_paid_meta,         -- creator_economy / paid_meta
  ltv_ratio_vs_affiliate_program, -- creator_economy / affiliate_program
  ltv_ratio_vs_tiktok_shop,       -- creator_economy / tiktok_shop
  -- 90-day-renewal-cadence-trigger (per 90-day-content-licensing-renewal-cadence)
  renewal_decision                -- 'renew' if avg_ltv >= 2x baseline; 'sunset' if avg_ltv < 0.5x baseline; 'hold' otherwise
FROM triple_whale.creator_cohort_ltv_5way_comparison
WHERE cohort_date BETWEEN '2026-01-01' AND CURRENT_DATE
ORDER BY cohort_date DESC, cohort_type, ltv_window;
```

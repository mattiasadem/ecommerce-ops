## The 5 voice profiles (canonical from `assets/02-brand-voice.md`)

The canonical 5-voice framework ships in `assets/02-brand-voice.md` §The 5-dimension voice framework. Reuse it verbatim for every TikTok-Shop-creator-brief + LIVE-show-runner-script + creator-payout-contract + product-listing-optimization-checklist + Shop-Score-4.8+-audit + FTC-compliance-disclosure-language piece:

- **Default** — confident + plain + lightly irreverent. Use for **mass-market DTC** voice brands.
- **Luxury** — refined + quiet + maximalist grammar. Use for **luxury DTC** voice brands.
- **Sustainable** — warm + earnest + educational. Use for **clean-beauty / sustainable** voice brands.
- **Gen-Z** — high-energy + lowercase + meme-aware. Use for **Gen-Z** voice brands.
- **B2B** — precise + benefit-led + no jokes. Use for **professional / hospitality / wellness-practitioner** retail buyers (the canonical default wholesale-buyer voice regardless of brand-DTC-voice).

**TikTok-Shop-creator-briefs rule:** the **creator-brief defaults to Gen-Z** (the canonical creator-affiliate voice on TikTok per TikTok for Business 2024 + Aspire 2024 benchmarks — 70%+ TikTok users 18-34 + creators skew Gen-Z) regardless of brand-DTC-voice. The 5-voice columns below show how to ADAPT the creator-brief when the brand's DTC voice carries through (e.g. a Luxury-DTC brand writing creator-briefs still sounds Luxury, not Gen-Z-default — the Luxury-creator-brief uses lowercase-minimalist-grammar + hand-written-thank-you-notes + ceremony-of-the-product + ultra-premium-margins per the 5-voice framework). For most Path-B brands, the creator-briefs read Gen-Z-default (high-energy + lowercase + meme-aware + 6-15-second-hook-payload-format per TikTok-LIVE-2024) with a brand-DTC-voice-overlaid creator-brief variant for the per-voice customer-segment.

## The 5-segment LIVE-show-runner-script-template (Path B DEFAULT 4-hour-week cadence)

The canonical 5-segment LIVE-show-runner-script runs **1 LIVE-session-per-week 60-90-minutes per session per TikTok-LIVE-2024-benchmarks** with the following segment-by-segment timing-budget + teleprompter-prompt-card + LIVE-segment-timer-30-second-warning-system. The script defaults to Gen-Z-Beauty (the canonical Path B TikTok-Shop profile) with the same 5-voice override columns from the creator-briefs above + per-SKU-archetype Segment-2-demo emphasis per the 6-SKU-archetype matrix.

### LIVE-show-runner-script (60-minute Path B baseline cadence — adjust +30 minutes for 90-minute sessions)

**Segment 1 — Product-intro with hero-SKU + opening-15-minute-frame [15 minutes]**

- **Teleprompter-prompt-card-1:** "Hey TikTok! It's [Your-Name] from [Brand], we're LIVE on the TikTok Shop with our [Hero-SKU]. Tell us where you're watching from in the chat! We've got a 25%-off-flash-sale at minute 50 — don't miss it. Tap the yellow basket to claim yours."
- **LIVE-segment-timer:** 30-second-warning at minute 14:30 → "We're moving to demo in 30 seconds — top up your basket!"
- **Voice-overrides per creator-voice:**
    - **Default:** plain + confident + product-intro-direct-sell.
    - **Luxury:** refined + quiet + "We are honoured to share [Hero-SKU] from our [Origin] atelier — a single, deliberate unveiling..."
    - **Sustainable:** earnest + warm + 5-ingredient-transparency-disclosure + carbon-neutral-shipping-pull-quote.
    - **Gen-Z:** high-energy + lowercase + "ok besties, we're LIVE, the basket is open, who's hyped? 💖"
    - **B2B:** precise + benefit-led + trade-buyer-pitch + wholesale-line-sheet-CTA.
- **Per-SKU-archetype Segment-1-emphasis:** Beauty = product-unboxing-ritual + ingredient-list-disclosure. Fashion = outfit-of-the-day-montage. Home = room-context-shoot. Food = first-bite-ritual. Pet = pet-meets-product. Health = FDA-disclaimer + routine-context.
- **First-LIVE-chat-engagement tactics:** respond to first 5-10 chat messages by name + tag product in basket + pin-product-link in LIVE-overlay. **Per TikTok-LIVE-2024 benchmarks, brands with first-5-min-engagement have 2-3× chat-engagement-rate vs brands that ignore first 5 minutes.**

**Segment 2 — Demo (30 minutes for 60-minute show / 45 minutes for 90-minute show)**

- **Teleprompter-prompt-card-2:** "Let's get into the demo. Watch [Demonstration-Step-1], [Demonstration-Step-2], [Demonstration-Step-3]. The texture is [Texture-Description]. The result is [Result-Description]. Type 'DEMO' in the chat if you want me to do another one on a specific request."
- **LIVE-segment-timer:** 30-second-warning at Segment-2-end → "We're moving to Q&A in 30 seconds — drop your questions NOW!"
- **Voice-overrides:** same 5 voices as Segment 1.
- **Per-SKU-archetype Segment-2-emphasis:** Beauty = texture-application-routine (the canonical 6-step routine: cleanse + tone + treat + moisturize + SPF + aftercare). Fashion = try-on-montage. Home = before-after-reveal. Food = taste + texture + recipe-application. Pet = pet-reaction + happy-dance. Health = routine-stack (4-product-stack) + FDA-disclaimer-reinforced.
- **Live-demo-fail-safe:** if the demo doesn't land (e.g. product texture is unexpected, pet doesn't cooperate, lighting blows out), the script defaults to "OK, that didn't go as planned — let's try it again together, [Brand] is sending the first 5 chat-engagers a free [Product] to make up for it." Per TikTok-LIVE-2024, brands that fail-and-recover transparently have +15-25% replay-rate vs brands that abort the LIVE.

**Segment 3 — Q&A with chat-responder [10 minutes for 60-minute show / 15 minutes for 90-minute show]**

- **Teleprompter-prompt-card-3:** "Drop your questions in the chat. Our community-team [Name-1] + [Name-2] are handling the questions while I focus on the demo. Top questions so far: [Q1], [Q2], [Q3]."

Home = installation + dimensions + assembly. Food = ingredients + dietary-restrictions + storage. Pet = breed-suitability + age-appropriateness + ingredients. Health = **FDA-required-disclaimer re-stated** + ingredient-questions + physician-recommendations.
